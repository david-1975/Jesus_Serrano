HTML Listas

Etiquetas de lista de datos

Una tabla HTML se define con la <table>etiqueta. Cada fila de la tabla se define con la <tr>etiqueta. Un encabezado de la tabla se define con la <th>etiqueta. Por defecto, encabezados de la tabla están en negrita y centrado. A los datos / celda de la tabla se define con la <td> etiqueta y pueden contener todo tipo de elementos HTML; texto, imágenes, listas, otras tablas, etc...

HTML ofrece varios mecanismos para especificar listas de información. Todas las listas deben contener uno o más objetos de lista. Las listas pueden contener: información no ordenada, información ordenada, y lista de descripciones.

Etiqueta Descripción
<ul> Define una lista desordenada
<ol> Define una lista ordenada
<li> Define un elemento de una lista
<dl> Define una lista de definiciones
<dt> Define un término en una lista de definiciones
<dd> Define una descripción en una lista de definición

Nota: La etiqueta <caption> define un título de la tabla y debe insertarse inmediatamente después de la <table> etiqueta. Sólo se puede especificar una leyenda (título) por tabla.


Tipos de listas

HTML ofrece varios mecanismos para especificar listas de información. Todas las listas deben contener uno o más objetos de lista. Las listas pueden contener: información no ordenada, información ordenada, información sobre definiciones.

Los tippos de listas en HTML son los siguientes:

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemneto 4

Lista ordenada

  1. Elemento

  3. Elemento
  4. Elemento

Lista descripción

Titulo 1
Elemento 1

Elemento 2

Lista anidada

  1. Elemento
    • Elemento 1
    • Elemento 1
  2. Elemneto

Listas desordenadas

Una lista no ordenada es un conjunto de elementos relacionados entre sí pero para los que no se indica un orden o secuencia determinados. La etiqueta <ul> encierra todos los elementos de la lista y la etiqueta <li> cada uno de sus elementos. Se puede utilizar el estilos CSS list-style-type para especificar el tipo de viñeta. El atributo type en la entiqueta <ul> no se admite en HTML5.


Lista type disc

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemneto 4

Lista type circle

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemneto 4

Lista type square

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemneto 4

Lista type none

  • Elemento 1
  • Elemento 2
  • Elemento 3
  • Elemneto 4

Listas Ordenadas

Las listas ordenadas son casi idénticas a las listas no ordenadas, salvo que en este caso los elementos relacionados se muestran siguiendo un orden determinado. La lista ordenada se define mediante la etiqueta <ol>. Los elementos de la lista se definen mediante la etiqueta <li>, la misma que se utiliza en las listas no ordenadas. Se puede utilizar el estilos CSS list-style-type para especificar el tipo de numeración. El atributo type en la entiqueta <ol> si se admite en HTML5.

Lista type="1"

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
  4. Elemneto 4

Lista type="A"

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
  4. Elemneto 4

Lista type="I"

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
  4. Elemneto 4

Lista start="10" reversed

  1. Elemento 1
  2. Elemento 2
  3. Elemento 3
  4. Elemneto 4

 

Listas de Descripción

HTML también soporta listas de descripción. Una lista de descripción es una lista de términos, con una descripción de cada término. La etiqueta <dl> define una lista de descripción, la etiqueta <dt> define el término (nombre), y la etiqueta <dd> describe cada término.

Lista type disc

Titulo 1
Elemento 1
Título 2
Elemento 2

Lista type circle

Titulo 1
Elemento 1
Título 2
Elemento 2

Lista type square

Titulo 1
Elemento 1
Título 2
Elemento 2

Lista type none

Titulo 1
Elemento 1
Título 2
Elemento 2

 

En HTML 4.01, la etiqueta <dl> se define como una lista de definiciones, y en HTML 5, la etiqueta <dl> se define como una lista de descripciones.

Las listas de definiciones o descripciones son similares a las de un diccionario. Cada elemento de la lista está formado por un termino y una definición. La etiqueta <dl> crea la lista de definición, y las etiquetas <dt> y <dd> definen respectivamente el término y la descripción de cada elemento de la lista.
