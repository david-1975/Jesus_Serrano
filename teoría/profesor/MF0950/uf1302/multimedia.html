Etiquetas multimedia

El HTML5 <video> elemento especifica una forma estándar para incrustar un vídeo en una página web

Etiqueta Descripción
<video> Define un vídeo
<source> Define una fuente de vídeo o audio
<track> Define un archivo de texto para subtítulos
<audio> Define una lista de definiciones
<iframe> Define un término en una lista de definiciones
<object> Define una descripción en una lista de definición
<embed> Define una descripción en una lista de definición

Ejemplo de uso VIDEO

Código HTML

<video width="320px" height="240" autoplay controls preload="auto" loop poster="img/paisaje.jpg">
<source src="video/mov_bbb.mp4" type="video/mp4">
<source src="videos/mov_bbb.webm" type="video/webm">
<source src="videos/mov_bbb.ogv" type="videos/ogg">
<track src="subtitulos_en.vtt" kind="subtitulos" 
srclang="en" label="English">
Tu navegor no soporta el elemento audio.
</video>

HTML Audio

HTML ofrece varios mecanismos para especificar listas de información. Todas las listas deben contener uno o más objetos de lista. Las listas pueden contener: información no ordenada, información ordenada, información sobre definiciones.

Ejemplo de uso AUDIO

Código HTML

<audio controls>
<source src="audio/intro.ogg" type="audio/ogg">
<source src="audio/intro.mp3" type="audio/mpeg">
<source src="audio/intro.mp3" type="audio/wav">
Tu navegor no soporta el elemento audio.
</audio>

HTML Object

El <object>elemento es soportado por todos los navegadores.

El <object>elemento define un objeto incrustado dentro de un documento HTML.

Ejemplo de uso Object

Código HTML

<object width="400" height="50" data="swf/bookmark.swf"></object>



Nota: Se puede utilizar el estilos CSS "list-style-type" para especificar el tipo de viñeta. El atributo type en la etiqueta <ul> no se admite en HTML5. Los tipos de viñeta admintidas en CSS son: circle: círculo negro y relleno, disc: línea círcular, square: cuadro relleno, none: Este valor elimina la viñeta.

Listas de definición

Las listas de definiciones o descripciones son similares a las de un diccionario. Cada elemento de la lista está formado por un termino y una definición. La etiqueta <dl> define una lista de descripción, la etiqueta <dt> define el término (nombre), y la etiqueta <dd> describe cada término.

Lista con margen

Término 1
Definición 1
Término 2
Definición 2

Lista sin margen

Término 1
Definición 1
Término 2
Definición 2

Lista invertida

Término 1
Definición 1
Término 2
Definición 2

Lista dt, dd, dd, dd

Término 1
Definición 1
Definición 2
Definición 3

Nota: Los navegadores formatean las listas de definición de forma similar a las otras listas, tabulando la definición y alineando a la izquierda los términos. Aunque no es habitual, cada término puede tener asociada más de una definición y cada definición puede tener asociada varios términos.
