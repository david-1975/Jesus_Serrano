El BOM (Modelo de Objetos del Navegador)

El Modelo examinador de objetos (BOM) permite Javascript para "hablar con" el navegador.

El Browser Object Model o BOM, permite acceder y modificar las propiedades de las ventanas del propio navegador. Mediante BOM, es posible redimensionar y mover la ventana del navegador, modificar el texto que se muestra en la barra de estado y realizar muchas otras manipulaciones no relacionadas con el contenido de la página HTML.

No existen normas oficiales para el Browser Object Modelo (BOM). El mayor inconveniente de BOM es que, al contrario de lo que sucede con DOM, ninguna entidad se encarga de estandarizarlo o definir unos mínimos de interoperabilidad entre navegadores.

Objetos de Pantalla

Método Descripción  
screen.width , screen.height Ancho y alto de la pantalla
screen.availWidth, screen.availHeight Ancho y alto de de la pantalla disponible
screen.colorDepth, screen.pixelDepth Profundidad de color y píxeles de la pantalla

Objetos de ventana

Método Descripción  
window.location.href  Devuelve el href (URL) de la página actual.
window.location.hostname Devuelve el nombre de dominio del proveedor de alojamiento web.
window.location.pathname Devuelve la ruta y el nombre de la página actual.
window.location.protocol Devuelve el protocolo web utilizado (http: o https:)
window.location.assign Carga un nuevo documento.
window.location.port Devuelve el puerto de conexión

Historial del navegador

Método Descripción  
history.back ( ) El método history.back ( ) carga la anterior URL en la lista del historial.
history.forward ( )  El método history.forward ( ) carga la siguiente URL en la lista del historial.
history.go ( )  El método history.go ( ) permite volver varias páginas hacia atrás (-2) o hacia adelante (2) en el historial.
history.length  La propiedad history.legth devuelve el número de URL en la lista del historial de la ventana del navegador actual.

Ventanas emergentes del navegador

Método Descripción  
window.alert("sometext"); Muestra un cuadro de alerta y el usuario tendrá tendrá que hacer clic en "Aceptar" para continuar.
window.confirm("sometext"); Muestra un cuadro de confirmación dónde el usuario tendrá que verificar o aceptar algo.
window.prompt("sometext","defaultText"); Muestra un cuadro con un mensaje para que el usuario introduzca un valor.

Eventos de tiempo en el navegador

Método Descripción  
window.setTimeout(function, milliseconds);
window.clearTimeout(timeoutVariable); 		El método setTimeout ( ) ejecuta una función en un intervalo de tiempo especificado.
El método clearTimeout ( ) detiene la ejecución de la función especificada en setTimeout ( ).
window.setInterval(function, milliseconds);
window.clearInterval(timerVariable); 		El método setInterval ( ) repite una función dada en cada intervalo de tiempo dado..
El método clearInterval ( ) detiene la ejecución de la función especificada en setTimeout ( ).
