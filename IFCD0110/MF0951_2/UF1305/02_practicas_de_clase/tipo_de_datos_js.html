Tipos de datos en JavaScript

El concepto de tipos de datos

Las variables de JavaScript pueden contener muchos tipos de datos: números, cadenas de texto, y más...

En la programación, el tipo de datos es un concepto importante, que nos permiten operar con las variables.

Sin tipos de datos, un ordenador no puede resolver con seguridad esto:

  • var a = 16 + "Volvo";
    a =

  • var b = "16" + "Volvo";
    b =

  • var c = "Volvo" + 16;
    c =
  • var d = 16 + 4 + "Volvo";
    d =

  • var e = "Volvo" + 16 + 4;
    e =

  • var f = 16 + "Volvo" + 4;
    f =
  • var g = "16" + 4 + "Volvo";
    g =

  • var h = "16" + "4" + "Volvo";
    h =

  • var i = "Volvo" + 16 + "4";
    i =

Tienes que saber...

  • JavaScript evalúa las expresiones de izquierda a derecha.
  • Al añadir un número y una cadena, JavaScript tratará el número como una cadena.
  • El signo + se utiliza para concatenar o unir valores, excepto cuando se trata de números, que es utilizado para sumar.

El operador typeof

En JavaScript typeof es un operador para mostrar el tipo de una variable. El operador typeof devuelve el tipo de una variable o de una expresión.

Typeof = String

Un string (cadena de texto) es uno o más caracteres escritos entre comillas simples o dobles. Las cadenas JavaScript se utilizan para almacenar y manipular texto.

Se puede usar comillas simples dentro de una cadena, siempre y cuando no coincidan con las comillas dobles que rodean la cadena.

Ejemplo de cadenas de texto

var coche1 = " ";
var coche2 = "Volvo";
var coche3 = "Volvo XC60";
var coche4 = "Volvo 'modelo' XC60";
var coche5 = "Volvo \"modelo\" XC60";

document.getElementById("demo").innerHTML = typeof (coche);

Ejemplo de cadenas con números

var numero1 = "10";
var numero2 = 10.50;
var numero3 = "123e5";
var numero4 = "19:33";
var numero5 = 654874587;

document.getElementById("demo").innerHTML = typeof (numero);

Typeof = Number

JavaScript tiene sólo un tipo de número. Los números se pueden escribir con o sin decimales.

Los decimales se separan con un punto, y no con una coma. Los números se escriben sin comillas.

Ejemplo de datos numéricos

var numerico1 = 3;
var numerico2 = 3.14;
var numerico3 = 123e5;
var numerico4 = 123e-5;
var numerico5 = 0xFF;

document.getElementById("demo").innerHTML = typeof (numerico);

Ejemplo de datos numéricos

var numeral1 = "10" - "5";
var numeral2 = '"10" / "5"';
var numeral3 = "10" * "5";
var numeral4 = (0.2 * 10 ) / 10;
var numeral5 = (0.2 * 10 + 0.1 * 10) / 10; 

document.getElementById("demo").innerHTML = typeof (numeral);

Typeof = Booleans, Object, Funtion

  • Los datos booleanos se utilizan para realizar comparativa entre datos y puede tener dos valores: true (verdadero) o false (falso).
  • Los objetos de datos se escriben con llaves y las propiedades del objeto se separan por comas. Las propiedades y sus valores se separan por dos puntos.
  • Las matrices de datos se escriben con corchetes y los elementos de la matriz están separados por comas. Son variables con múltiples datos.
  • Una función es un bloque de código diseñado para realizar una tarea en particular que se ejecuta cuando se la invoca (llama).

Ejemplo de datos Booleanos

var x = 5; var y = 5; var z = 6;
comparacion = (x == y) ;   // Retorna true (verdadero)
comparacion = (x == z);   // Retorna false (falso)

document.getElementById("demo").innerHTML = typeof (comparacion);

Ejemplo de datos Función

function miFuncion (x, y)
{ var x = 3; var y = 5; var z = ( x * y );
return z; };

document.getElementById("demo").innerHTML = typeof (miFuncion);
