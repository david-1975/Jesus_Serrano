Estructuras de control de flujo

El concepto de control de flujo

Con las estructuras de control de flujo, los programas dejan de ser una sucesión lineal de instrucciones para convertirse en programas inteligentes que pueden tomar decisiones en función del valor de las variables. Son instrucciones del tipo "si se cumple esta condición, haz esto; si no se cumple, haz esto otro". También existen instrucciones del tipo "repite esto mientras se cumpla esta condición".

Las estructuras de control definen el flujo de ejeccución del programa, y con ellas podemos resolver cualquier logaritmo. Básicamente son tres: secuencial, condicional y repetitiva, aunque cada una de ellas admite ciertas variantes y distintas sintaxis.

Estructura Secuencial

Se compone de instrucciones que se ejecutan secuencialmente, una detrás de la otra, sin ningún tipo de condición ni situación que impida su ejecucución de principio a fin del script.

Estructura Condicional

Son instrucciones con una condición, si al evaluar la condición la respuesta es verdadera se ejecutarán una serie de instrucciones, y si es falsa se ejecutarán otras.

Estructura Repetitiva

Se utilizan cuando se quiere que un conjunto de instrucciones se ejecuten un cierto número finito de veces. Se les llama bucle o ciclo a todo proceso que se repite cierto número de veces.

Estructura condicional

Las sentencias condicionales se utilizan para llevar a cabo diferentes acciones basadas en diferentes condiciones.

Método Descripción Sintaxis  
if Ejecuta una serie de instrucciones si se cumple una determinada condición. if (condicion) {
Bloque de código
}
else Ejecuta una segunda serie de instrucciones si no se cumple una determinada condición. if (condicion) {
Bloque de código
} else {
Bloque de código
}
esle if Especifica una nueva condición si la primera condición no se cumple.

if (condicion) {
Bloque de código
} else if (condicion) {
Bloque de código
} else {
Bloque de código }

switch Realiza diferentes acciones basadas en diferentes condiciones. switch(expresión) {
case a:  Bloque de código        break;
case b:  Bloque de código        break;
case z:   Bloque de código;
default:   Bloque de código
}

Estructura repetitiva

Los bucles son muy útiles, si desea ejecutar el mismo código una y otra vez, cada vez con un valor diferente.

Método Descripción Sintaxis  
for Ejecuta un bloque de código varias veces, cada vez con un valor diferente. for (declaración 1; declaración 2; declaración 3) {
    Bloque de código a ejecutar
}
while Ejecuta un bloque de código, siempre que la condición especificada sea verdadera. while (condición) {
    Bloque de código a ejecutar
}
do while Ejecuta el bloque de código una vez, antes de comprobar si la condición es verdadera, y se repite el bucle mientras la condición es verdadera. do {
    code block to be executed
}
while (condition);

Sentencias saltar y continuar

La sentencia break "salta" de un bucle. La sentencia continue "salta sobre" una iteración en el bucle.

Método Descripción Sintaxis  
break Rompe un bucle, si se produce una condición especificada, y continúa con la siguiente iteración del bucle. break labelname; 
continue Continua un bucle, si se produce una condición especificada, y continúa con la siguiente iteración del bucle. continue labelname;
