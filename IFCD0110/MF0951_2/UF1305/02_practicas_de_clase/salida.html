Salida Javascript

Las posibilidades de visualización de JavaScript

JavaScript puede alterar los datos "Display" de diferentes maneras:

  • La escritura en un elemento HTML, usando innerHTML.
  • La escritura en la salida HTML utilizando document.write().
  • Escribir en un cuadro de alerta, utilizando window.alert().
  • Escribir en la consola del navegador, usando console.log().

usando innerHtml

Define el contenido HTML

<button onClick="demo1()" class="btn-primary">Pulsa</button>
<p id="demo1"></p>
<script>
function demo1 () {
document.getElementById("demo1").innerHTML = 5+6;
}
</script>

usando document.write

Se utiliza para pruebas

<button onClick="demo2()" class="btn-primary">Pulsa</button>
<script>
function demo2 () {
document.write(5+6);
}
</script>

usando window.alert

Saca un cuadro de alerta

<button onClick="demo3()" class="btn-primary">Pulsa</button>
<script>
function demo3 () {
window.alert(5+6);
}
</script>

usando console.log

Muestra en consola

<button onClick="demo4()" class="btn-primary">Pulsa</button>
<script>
function demo4 () {
console.log(5+6);
}
</script>


Cuadros de diálogos de ventanas

JavaScript puede mostrar cuadros de diálogos a través del DOM "Window" de diferenteas maneras:
  • El metodo confirm() muestra un cuadro diálogo con un mensaje y la oción de aceptar o cancelar
  • El metodo alert() muestra un cuadro de diálogo con un mensaje de alerta.
  • El metodo prompt() muestra un cuadro de diálogo solicitando al usuario entrada de datos.

confirm()

Muestra un cuadro de diálogo con un mensaje especificado, junto con un clic en Aceptar y un botón Cancelar

confirm(message)

alert()

Muerstra un cuadro de alerta con un mensaje especificado y un botón Aceptar

 

alert(message)

prompt()

Muestra un cuadro de diálogo que solicita al visitante una entrada de datos

 

prompt(text, defaultText)

moveBy() y moveTo()

moveBy() mueve una ventana creada a una posición determinada y moveTo() mueve una ventana creada los parámetros indicado

moveBy(x,y) moveTo(x,y)

resizeTo() y resizeBy()

resizeTo() cambia el tamaño de una ventana a un tamaño específico y resizeBy() agranda o recorta una ventana

resizeBy(x,y) resizeBy(x,y)
