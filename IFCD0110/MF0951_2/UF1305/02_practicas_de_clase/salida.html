Salida Javascript

Las posibilidades de visualización de JavaScript

JavaScript puede alterar los datos "Display" de diferentes maneras:

  • La escritura en un elemento HTML, usando innerHTML.
  • La escritura en la salida HTML utilizando document.write().
  • Escribir en un cuadro de alerta, utilizando window.alert().
  • Escribir en la consola del navegador, usando console.log().

usando innerHtml

Define el contenido HTML

<button onClick="demo1()" class="btn-primary">Pulsa</button>
<p id="demo1"></p>
<script>
function demo1 () {
document.getElementById("demo1").innerHTML = 5+6;
}
</script>

usando document.write

Se utiliza para pruebas

<button onClick="demo2()" class="btn-primary">Pulsa</button>
<script>
function demo2 () {
document.write(5+6);
}
</script>

usando window.alert

Saca un cuadro de alerta

<button onClick="demo3()" class="btn-primary">Pulsa</button>
<script>
function demo3 () {
window.alert(5+6);
}
</script>

usando console.log

Muestra en consola

<button onClick="demo4()" class="btn-primary">Pulsa</button>
<script>
function demo4 () {
console.log(5+6);
}
</script>
