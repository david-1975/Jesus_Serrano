Métodos fechas y hora (Date)

El concepto de fecha

Por defecto, JavaScript utilizará la zona horaria del navegador y mostrar una fecha como una cadena de texto completo:

Mar Mar 19 de 2019 12:50:43 GMT + 0100 (Hora Estándar de Europa central)

Los métodos numéricos devuelven un valor nuevo y no cambian la variable original, y puede utilizarse con cualquier tipo de número (literales, variables o expresiones).

Fecha objetos se crean con el new Date()constructor. new Date()crea un nuevo objeto Date con la fecha y hora actual.

Métodos fechas

Método Descripción  
getDate() Crea un nuevo objeto Date con la fecha y hora actual
getFullYear() Comprueba si un valor es un entero.
getMonth() Comprueba si un valor no es un número (NaN).
getHours() Convierte un número en un número exponencial. x es igual al número de dígitos.
getMinutes() Formatea un número con x números de dígitos después del punto decimal.
getSeconds() Formatea un número a una longitud.
getMilliseconds() Convierte un número en una cadena y lo representa en una base aritmética.
getTime() Devuelve el valor primitivo de un número.
getDay() Devuelve el valor primitivo de un número.
Date.now() Devuelve el valor primitivo de un número.
