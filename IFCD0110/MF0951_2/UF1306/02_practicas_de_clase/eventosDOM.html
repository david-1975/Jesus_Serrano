Eventos

Se llama evento a un suceso que tiene lugar en el sistema cuando un usuario ejecuta alguna acción. Los eventos se asocian a elementos HTML y las acciones que se ejecutan se programan en JavaScript . JavaScript puede ejecutarse cuando se produce un evento, como cuando un usuario hace clic en un elemento HTML. Para ejecutar el código cuando un usuario hace clic en un elemento, se añade código JavaScript a un atributo de evento HTML.

Sintaxis: onclick="JavaScript";

Eventos de ratón

Estos tipos de eventos pertenece al objeto MouseEvent:

Evento Descripción
onclick El evento ocurre cuando el usuario hace clic en un elemento.
oncontextmenu El evento ocurre cuando el usuario hace clic con el botón derecho en un elemento para abrir un menú contextual.
ondblclick El evento ocurre cuando el usuario hace doble clic en un elemento.
onmousedown El evento ocurre cuando el usuario presiona un botón del mouse sobre un elemento.
onmouseenter El evento ocurre cuando el puntero se mueve a un elemento.
onmouseleave El evento ocurre cuando el puntero se mueve fuera de un elemento.
onmousemove El evento ocurre cuando el puntero se mueve mientras está sobre un elemento.
onmouseout El evento ocurre cuando un usuario mueve el puntero del mouse fuera de un elemento o de uno de sus hijos.
onmouseover El evento ocurre cuando el puntero se mueve a un elemento o a uno de sus hijos.
onmouseup El evento ocurre cuando un usuario suelta un botón del mouse sobre un elemento.

Eventos del Portapapeles

Estos tipos de eventos pertenece al objeto ClipboardEvent:

Event Description Funcionamiento
oncopy El evento ocurre cuando el usuario copia el contenido de un elemento.
oncut El evento ocurre cuando el usuario corta el contenido de un elemento.
onpaste El evento ocurre cuando el usuario pega el contenido de un elemento.

Eventos de teclado

Estos tipos de eventos pertenece al objeto KeyboardEvent:

Evento Descripción  
onkeydown El evento ocurre cuando el usuario está presionando una tecla.
onkeypress El evento ocurre cuando el usuario presiona una tecla.
onkeyup El evento ocurre cuando el usuario libera una tecla.
