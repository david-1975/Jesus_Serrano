Multimedia viene en muchos formatos diferentes. Puede ser casi cualquier cosa que usted puede escuchar o ver.

Ejemplos: Imágenes, música, sonido, vídeos, discos, películas, animaciones y mucho más.

Las páginas Web a menudo contienen elementos multimedia de diferentes tipos y formatos.

Etiqueta Descripción
<video> Define un vídeo en una página web
<source> Define una fuente de vídeo o audio
<track> Define un archivo de texto para subtítulos
<audio> Define un audio en una página web
<iframe> Define un elemento externo en una página web
<object> Define un objeto incrustado dentro de un documento HTML
<embed> Tambien define un objeto incrustado dentro de un documento HTML

HTML Video

La etiqueta <video> especifica una forma estándar para incrustar un vídeo en una página web.

El atributo controls agrega controles de vídeo, como reproducción, pausa y volumen.

Es una buena idea incluir siempre los atributos width y height. Si la altura y anchura no están establecidos, la página puede parpadear mientras se carga el vídeo.

El <source>elemento le permite especificar los archivos de vídeo alternativas que el navegador puede elegir. El navegador usará el primer formato reconocido.

El texto entre el <video> y </video> etiquetas sólo se mostrará en los navegadores que no soportan el <video> elemento.

Ejemplo de uso

<video width="320" height="240" controls preload="auto" autoplay poster="img/comodo-dragon.jpg">
<source src="video/mov_bbb.mp4" type="video/mp4">
<source src="video/mov_bbb.ogg" type="video/ogg">
<source src="video/mov_bbb.ogg" type="video/ogg">
<track src="subtitulos_en.vtt" kind="subtitulos" srclang="en" label="English">
</video>
<video src="video/mov_bbb.mp4" width="320" height="240" controls poster="img/comodo-dragon.jpg">
</video>

Audio

La etiqueta <audio> especifica una forma estándar para incrustar un audio en una página web.

El atributo controls agrega controles de audio, como reproducción, pausa y volumen.

El elemento <source> le permite especificar los archivos de audio alternativos que el navegador puede elegir. El navegador usará el primer formato reconocido.

El texto entre las etiquetas <audio> y </audio> sólo se mostrará en los navegadores que no soportan el elemento <audio>.

Ejemplo de uso

<audio controls>
<source src="audio/intro.mp3" type="audio/mp3">
<source src="audio/intro.ogg" type="audio/ogg">
<source src="audio/intro.mp3" type="audio/wav">
</audio>

HTML Object

El <object>elemento es soportado por todos los navegadores.

El <object>elemento define un objeto incrustado dentro de un documento HTML.

Ejemplo de uso

<object width="400" height="50" data="bookmark.swf"></object>
