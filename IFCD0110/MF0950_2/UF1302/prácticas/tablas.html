HTML Tablas

Etiquetas que definen una tabla

Una tabla HTML se define con la etiqueta<table>.

Cada fila de la tabla se define con la etiqueta<tr>. Un encabezado de la tabla se define con la etiqueta<th>. Por defecto, encabezados de la tabla están en negrita y centrado. A los datos / (cierre de etiqueta) celda de la tabla se define con la etiqueta<td>.

Etiqueta Descripción
<table> Define una tabla
<tr> Define una fila en una tabla
<th> Define una celda de cabecera en una tabla
<td> Define el contenido de las celdas
<caption> Define el título o leyenda de una tabla
<colgroup> Especifica en una tabla un grupo de una o más columnas para darle formato
<col> Especifica propiedades de columna para cada columna dentro de un elemento <colgroup>
<thead> Agrupa el contenido del encabezado en una tabla
<tbody> Agrupa el contenido del cuerpo en una tabla
<tfoot> Agrupa el contenido del pie de página en una tabla

El atributo Scope

El atributo scope provee un mecanismo para indicar explícitamente a que celdas afecta una celda de encabezado (sólo en las etiquetas <th>). El atributo scope no tiene ningún efecto visual en los navegadores web normales, pero puede ser utilizado por los lectores de pantalla.Tiene un significado mayoritariamente sintáctico ya que el propio <th> ya indica que es encabezado de tabla.

Los atributos Colspan y Rowspan

El atributo colspan define el número de columnas de una célula debe ocupar.El atributo rowspan especifica el número de filas de una célula debe ocupar.

Colgroup y col

La etiqueta <colgroup> especifica un grupo de una o más columnas en una tabla para el formato. La etiqueta <colgroup> es útil para la aplicación de estilos para columnas completas, en lugar de repetir los estilos para cada célula, para cada fila.

La etiqueta <col> especifica las propiedades de columna para cada columna dentro de un <col> elemento. La etiqueta <col> es útil para la aplicación de estilos para columnas completas, en lugar de repetir los estilos para cada célula, para cada fila.
