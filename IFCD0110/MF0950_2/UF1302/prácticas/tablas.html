Etiquetas de estructura de datos en filas y columnas

Una tabla HTML se define con la <table>etiqueta. Cada fila de la tabla se define con la <tr>etiqueta. Un encabezado de la tabla se define con la <th>etiqueta. Por defecto, encabezados de la tabla están en negrita y centrado. A los datos / celda de la tabla se define con la <td> etiqueta y pueden contener todo tipo de elementos HTML; texto, imágenes, listas, otras tablas, etc...

Etiqueta Descripción
<table> Define una tabla
<th> Define una celda de encabezado en una tabla
<tr> Define fila en una tabla
<td> Define una celda en una tabla
<caption> Define un título de tabla
<colgroup> Especifica un grupo de una o más columnas en una tabla para formatear
<col> Especifica las propiedades de columna para cada columna dentro de un elemento <colgroup>
<thead> Agrupa el contenido del encabezado en una tabla.
<tbody> Agrupa el contenido del cuerpo en una tabla.
<tfoot> Agrupa el contenido del pie de página en una tabla.

Nota: La etiqueta <caption> define un título de la tabla y debe insertarse inmediatamente después de la <table> etiqueta. Sólo se puede especificar una leyenda (título) por tabla.

El atributo scope

El atributo scope especifica si una celda de encabezado es una cabecera de una columna, fila o grupo de columnas o filas. El atributo scope no tiene ningún efecto visual en los navegadores web normales, pero puede ser utilizado por los lectores de pantalla.

El atributo colspan y rowspan

El atributo colspan define el número de columnas de una celda que debe ocupar al espandirse las celdas y el atributo rowspan especifica el número de filas de una celda que debe ocupar. Sólo se escifica el atributo en las etiquetas <th> y <td>.

La etiqueta colgroup y col

La etiqueta <colgroup> especifica un grupo de una o más columnas en una tabla para la aplicación de estilos CSS para columnas completas, en lugar de repetir los estilos para cada celda, y para cada fila. Para definir propiedades diferentes a una columna dentro de un <colgroup>, utilice la etiqueta <col> dentro de la etiqueta <colgroup> junto con el atributo span para nombrar cuantas columnas se deben agrupar.

Agrupación por filas

HTML provee tres elementos que pueden ayudar a los autores a organizar la información presentada en una tabla. Éstos son: thead, para agrupar las filas que presentan información de encabezado; tfoot para contener a las filas que representan un pie o resumen; y tbody, para representar un bloque de filas que consiste en un cuerpo de datos.
