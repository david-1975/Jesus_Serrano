Las listas se utilizan para enumerar varios elementos en un documento y se pueden estructurar en 3 tipos: listas desordenadas, listas ordenadas y listas de definición.

Listas desordenadas

La etiqueta <ul> define una lista desordenada (viñetas). Utilice la etiqueta <ul> junto con la etiqueta <li> para crear listas desordenadas.Se puede utilizar el estilo CSS list-style-type para especificar el tipo de viñeta. El atributo type en la entiqueta <ul> no se admite en HTML5.

Listas ordenadas

La etiqueta <ol> define una lista ordenada. Una lista ordenada puede ser numérico o alfabético.

Utilice el <li> etiqueta para definir elementos de la lista. Al igual que en la listas desordenadas, se puede utilizar el estilo CSS list-style-type para especificar el tipo de viñeta.

Listas de definición

La <dl> etiqueta define una lista de descripción.

La etiqueta <dl> se utiliza en conjunción con <dt> (define términos / nombres) y <dd> (describe cada término / nombre).

Café
Es una bebida de color negro y se suele servir caliente.
Leche
Es una bebida de color blanco y se puede servir fría y caliente.

Etiquetas para formar listas


Etiqueta Descripción
<ul> Define una lista desordenada
<ol> Define una lista ordenada
<li> Define los elementos de las listas
<dl> Define una lista de definición
<dt> Define el título de la lista de definición
<dd> Describe el elemento del título de definición

Nota: La etiqueta <td> marca un elemento a definir y debe insertarse inmediatamente después de la etiqueta<dt>. Sólo se puede especificar una leyenda (título) por tabla.
