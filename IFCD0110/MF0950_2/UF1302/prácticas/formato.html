Formato HTML

El formato de texto, es el marcado que se aplica a un texto para añadirle datos de forma.

Al texto resultante se le conoce como texto formateado, texto con estilos o texto enriquecido.

Encabezados de texto

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.

Los usuarios ojean las páginas por sus títulos. Es importante utilizar los encabezados para mostrar la estructura del documento.

Los encabezamientos<h1> deben ser utilizados para los encabezados principales, seguido por los encabezamientos <h2>, <h3>, etc... que son menos importantes.

Nota: Usar encabezados HTML para sólo encabezados. No utilice encabezados para que el texto GRANDE o negrita.

Etiquetas que dan formato al texto

HTML define etiquetas especiales para la definición de texto con un especial significado.

HTML utiliza etiquetas para el formato de salida, como negrita o cursiva texto.

Etiqueta Descripción Ejemplo Utilización
<b> Define un texto en negrita Este texto está en negrita <b>Este texto está en negrita<b>
<i> Define un texto en cursiva Este texto está en cursiva <i>Este texto está en cursiva<i>
<strong> Define un texto en importante Este texto está resaltado <strong>Este texto está resaltado<strong>
<em> Define un texto enfatizado Este texto está enfatizado <em>Este texto está enfatizado<em>
<small> Define un texto más pequeño Este texto es más pequeño <Este texto essmall> más pequeño<small>
<sup> Define un texto en superíndice Este texto es unsuperíndice de texto <Este texto es unsup> superíndice<sup> de texto
<sub> Define un texto en subíndice Este texto es un subíndice de texto <Este texto es unsub> subíndice<sub> de texto
<ins> Define un texto insertado o agregado Este texto está insertado Este texto está <ins>insertado<ins>
<s> Define un texto tachado Este texto está tachado <s> Este texto está tachado<s>
<u> Define un texto subrayado Este texto está subrayado Este texto está <u> subrayado<u>
<del> Define un texto eliminado o borrado Este texto está tachado eliminado Este texto está<del>tachado<del> eliminado
<mark> Define un texto marcado Este texto está marcado <mark>Este texto está marcado<mark>

Nota: Los navegadores muestran <strong>como <b>, y <em>como <i>. Sin embargo, hay una diferencia en el significado de las siguientes etiquetas: <b>y <i>define el texto en negrita y cursiva, pero <strong>y<em> significa que el texto es "importante".
