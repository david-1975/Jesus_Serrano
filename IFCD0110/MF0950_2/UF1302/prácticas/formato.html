El formato del texto

El formato de texto, es el marcado que se aplica a un texto para añadirle datos de forma.

Al texto resultante se le conoce como texto formateado, texto con estilos o texto enriquecido.

Encabezados de texto

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.

Los usuarios ojean las páginas por sus títulos. Es importante utilizar los encabezados para mostrar la estructura del documento.

Los encabezamientos<h1> deben ser utilizados para los encabezados principales, seguido por los encabezamientos <h2>, <h3>, etc... que son menos importantes.

Nota: Usar encabezados HTML para sólo encabezados. No utilice encabezados para que el texto GRANDE o negrita.

Etiquetas que dan formato al texto

HTML define etiquetas especiales para la definición de texto con un especial significado.

HTML utiliza etiquetas para el formato de salida, como negrita o cursiva texto.

Etiquetas que separan el texto

La etiqueta <p>separa el texto, añadiendo automáticamente un espacio (margen) antes y después de cada elemento <p>. El navegador eliminará todos los espacios en blanco y las líneas adicionales cuando se visualiza la etiqueta <p>.

La etiqueta <br>es una etiqueta vacía, lo que siginifica que no tiene ninguna etiqueta final. Inserta un salto de línea (una nueva línea) sin comenzar un nuevo párrafo.

Un texto dentro de una etiqueta <pre> se muestra en una fuente de ancho fijo (generalmente Courier), y conserva ambos espacios y saltos de línea.
