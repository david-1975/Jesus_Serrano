Introducción a HTML

¿Qué es HTML?

HTML es el lenguaje de marcado estándar para la creación de páginas Web.

  • HTML significa Hyper Text Markup Language
  • HTML describe la estructura de las páginas Web usando marcado
  • Elementos HTML son los bloques de construcción de páginas HTML
  • Elementos HTML están representados por etiquetas
  • HTML etiquetas de la etiqueta de piezas de contenido como "mesa" "la partida", "párrafo", y así sucesivamente
  • Los navegadores no muestran las etiquetas HTML, pero las usan para representar el contenido de la página

Estructura básica de un HTML

<!DOCTYPE html>
<html lang="es-ES">
<head>
<title>Page Title</title>
<meta charset="UTF-8">
</head>
<body>
<h1>Encabezado de la página</h1>
<p>Párrafo de texto</p>
</body>
</html>

Explicación de las etiquetas básicas

  1. La etiqueta <!DOCTYPE html> define este documento para ser HTML5.
  2. La etiqueta <html> define el elemento raíz de una página HTML.
  3. El atributo lang="es-ES" especifica el idioma del documento y va dentro de la etiqueta <html>.
  4. La etiqueta <head> contiene información de metadatos sobre el documento.
  5. La etiqueta <title> especifica un título para el documento HTML.
  6. La etiqueta <meta charset="UTF-8"> define la codificación de caracteres.
  7. La etiqueta body> engloba el contenido de la página visible.
  8. La etiqueta <h1> define un encabezado.
  9. La etiqueta <p> define un párrafo.

Etiquetas HTML

Las etiquetas HTML son los nombres de elementos entre paréntesis angulares:

<Nombre de etiqueta> contenido va aquí ... </Nombre de etiqueta>

Etiquetas HTML
Normalmente vienen en pares como <p> y </p>.
Las etiquetas HTML
Están formadas por una etiqueta de inicio y una etiqueta de fin.
La etiqueta final
Se escribe igual que la etiqueta de inicio, pero con una barra inclinada insertada antes de que el nombre de la etiqueta.

Consejo: La etiqueta de inicio también se llama la etiqueta de apertura y la etiqueta final como etiqueta de cierre.

Navegadores web

El propósito de un navegador web (Chrome, IE, Firefox, Safari) es leer los documentos HTML y mostrarlos.
