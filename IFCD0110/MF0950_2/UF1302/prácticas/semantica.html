Elementos Semánticos

Un elemento semántico describe claramente su significado tanto para el navegador y el desarrollador.

Los ejemplos de no-semánticos elementos: <div> y <span> - dicen nada acerca de su contenido.

Ejemplos de semánticas elementos: <form>, <table> y <article> - definen claramente su contenido.

Etiqueta Descripción
<article> Define un articulo
<aside> Define un contenido añadido a la información de la página
<details> Define detalles adicionales que el usuario puede ocultar
<figcaption> Define un titulo para el elemento <figure>
<figure> Especifica contenido añadido de la información, normalmente imágenes
<footer> Define el pie de página de un documento o sección
<header> Define la cabecera
<main> Define el contenido principal de la página
<mark> Define texto marcado
<nav> Define links de navegación en la página
<section> Define una sección en un documento
<summary> Define la cabecera de un elemento <details>
<time> Define una fecha/hora
