HTML Formato

El formato del texto

El formato de texto, es el marcado que se aplica a un texto para añadirle datos de forma. Al texto resultante se le conoce como texto formateado, texto con estilos o texto enriquecido.

Encabezados de texto

Los motores de búsqueda utilizan los encabezados para indexar la estructura y el contenido de sus páginas web.

Los usuarios hojear sus páginas por sus títulos. Es importante utilizar los encabezados para mostrar la estructura del documento.

Los encabezamientos <h1> deben ser utilizados para los encabezados principales, seguido por encabezamientos <h2>, <h3>, que son menos importante.

Nota: Usar encabezados HTML para sólo encabezados. No utilice encabezados para que el texto GRANDE o negrita.

Etiquetas que dan forma al texto

HTML utiliza las etiquetas para la definición del texto con un especial significado.

HTML utiliza etiquetas para el formato de salida, como negrita o cursiva texto.

Etiqueta Descripción Visualización Utilización
<b> Define un texto en negrita Este texto está en negrita <b>Este texto está en negrita</b>
<i> Define un texto en cursiva Este texto está en cursiva <i>Este texto está en cursiva</i>
<strong> Define un texto importante Este texto es importante <strong>Este texto es importante</strong>
<em> Define un texto enfatizado Este texto está enfatizado <em>Este texto está enfatizado</em>
<small> Define un texto más pequeño Este texto es más pequeño Este texto es <small>más pequeño</small>
<sup> Define un texto superíndice Este texto está en superíndice Este texto es un <sup>superíndice</sup>
<sub> Define un texto subíndice Este texto está en subíndice Este texto es un <sub>subíndice</sub> de texto
<ins> Define un texto insertado Este texto está insertado agregado Este texto está insertado <ins>agregado</ins>
<s> Define un texto tachado Este texto está tachado Este texto es un texto <s>tachado</s>
<u> Define un texto subrayado Este texto está subrayado Este texto es un texto <u>subrayado</u>
<del> Define un texto borrado Este texto está borrado eliminado Este texto está <del>borrado</del> eliminado
<mark> Define un texto marcado Este texto ha sido marcado Este texto ha sido <mark>marcado</mark>

Nota: Los navegadores muestran <strong> como <b>, y <em>como <i>. Sin embargo, hay una diferencia en el significado de las siguientes etiquetas: <b> y <i> define el texto en negrita y cursiva, pero <strong>y <em>significa que el texto es "importante".

Etiquetas que separan el texto

La etiqueta <p> separa el texto, añadiendo automáticamente un espacio (margen) antes y después de cada elemento <p>.El navegador eliminará todos los espacios en blanco y las líneas adicionales cuando se visualiza la etiqueta <p>.

La etiqueta <br> es una etiqueta vacía, lo que significa que no tiene ninguna etiqueta final. Inserta un salto de línea (una nueva línea) sin comenzar un nuevo párrafo.

Un texto dentro de una etiqueta <pre> se muestra en una fuente de ancho fijo (generalmente Courier), y conserva ambos espacios y saltos de línea.
