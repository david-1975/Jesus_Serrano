Etiquetas que citan el texto

Las abreviaturas de marcado <abbr> pueden proporcionar información útil a los navegadores, sistemas de traducción y motores de búsqueda. Utiliza el atributo title para definir el significado de la abreviación.

La etiqueta de marcado <cite> hace referencia a un trabajo creativo como un libro, una película, un programa de televisión o una canción, no el nombre de una persona.

La etiqueta de marcado <blockquote> define una cita larga que se representa como un elemento de bloque. No genera automáticamente comillas alrededor del contenido.

La etiqueta de marcado <q> define una cita corta en línea. Marca las citas a otros autores o documentos. Genera automáticamente comillas alrededor del contenido.

La etiqueta de marcado <dfn> define un término y debe contener el atributo title para la definición / explicación del término dentro.

Etiquetas que agrupan el texto

El elemento <div> se utiliza a menudo como un contenedor para otros elementos HTML. El elemento <div> tiene atributos, pero no son necesarios. Los más comunes son style, class y id, para dar estilos CSS al bloque de contenido.

El elemento <span> se utiliza como un contenedor para agrupar un texto en línea. Un elemento en línea no se inicia en una nueva línea, sólo ocupa el mismo ancho que sea necesario. El elemento <span>, también tiene atributos, pero tampo son necesarios. Los más comunes son style, class y id, para dar estilos CSS al bloque de contenido.

Nota: Un elemento en bloque siempre empieza en una nueva línea y ocupa todo el ancho disponible (se extiende hacia la izquierda y la derecha tanto como sea posible). Un elemento en línea no se inicia en una nueva línea, sólo ocupa el mismo ancho que sea necesario. 

Etiquetas de código de computadora

  • La etiqueta <code> define un fragmento de código de ordenador.
  • La etiqueta <kbd> representa comandos de entrada del usuario.
  • La etiqueta <samp> representa la salida de un programa o sistema informático.
  • La etiqueta <var> se utiliza como expresión matemática o variable de la programación.
