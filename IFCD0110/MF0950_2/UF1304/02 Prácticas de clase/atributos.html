Atributos HTML5

HTML5 añadió los siguiente atributos para <input>:

  • autocomplete
  • autofocus
  • form
  • formaction
  • formenctype
  • formmethod
  • formnovalidate
  • formtarget
  • height and width
  • list
  • min and max
  • multiple
  • pattern (regexp)
  • placeholder
  • required
  • step

Los atributos de form los veremos más adelante

autocomplete

Especifica si un campo de formulario o de entrada debe tener autocompletar encendido o apagado. Si no ponemos nada, automáticamente estará en "on"

Atributo autocomplete

autofocus

Especifica que el campo de entrada de forma automática debe obtener el foco(puntero de escritura) cuando se carga la página.

Atributo autofocus

height and width

Especifican la altura y la anchura de un <input type="image">elemento

Atributos height y width:

list

Se refiere a un elemento <datalist> que contiene opciones predefinidas para un elemento <input>

Atributos list:

min y max

Especifican los valores mínimo y máximo para un elemento <input>. Trabajan con los siguientes tipos de entrada: número, rango, fecha, fecha y hora local, el mes, el tiempo y la semana.

multiple

Especifica que el usuario está autorizado para entroducir más de un valor en el <input>elemento. Trabaja con email y archivos.

Atributo multiple:

pattern (regexp)

Especifica una expresión regular que el valor <input> del elemento se compara. Trabaja con los siguientes tipos de entrada: texto, búsqueda, URL, teléfono, correo electrónico y contraseña. En esta web podemos ver varios tipos de patrones www.lenguajehtml.com

Atributo pattern: Codigo pais:

placeholder

Especifica una pista que describe el valor esperado de un campo de entrada (un valor de muestra o una breve descripción del formato).

Atributos placeholder:

required

Especifica que un campo de entrada debe ser completado antes de enviar el formulario.

Atributo required:

step

Especifica los intervalos de números para un elemento <input>. Trabaja con los siguientes tipos de entrada: número, rango, fecha, fecha y hora local, el mes, el tiempo y la semana.

Atributo step
