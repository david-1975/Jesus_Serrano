El elemento <input>

El elemento de formulario más importante es el elemento <input>. Se puede mostrar en varias formas, dependiendo del atributo type. Ejemplo: <input name="firstname" type="text"> Si el "type" se omite el atributo, el campo de entrada obtiene el tipo por defecto: "texto".

Etiqueta <label>

La etiqueta <label> define una etiqueta para un botón <>, <input>, <metro>, <salida>, <progreso>, <seleccionar> o elemento <textarea>. El elemento <label> no hace como algo especial para el usuario. Sin embargo, proporciona una mejora la facilidad de uso para los usuarios de ratón, ya que si el usuario hace clic en el texto dentro del elemento <label>, se alterna el control. El atributo de la etiqueta <label> debería ser igual al atributo id del elemento relacionado para unirse juntos.
