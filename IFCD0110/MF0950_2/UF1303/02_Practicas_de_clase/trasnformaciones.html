Las transformaciones CSS

La propiedad transforms aplica una transformación 2D o 3D a un elemento. Esta propiedad le permite rotar, escalar, mover, inclinación, etc., elementos.

Transformaciones CSS 2D

Métodos de transformación 2D:

  • translate(): translación
  • rotate(): rotación
  • scale(): tamaño
  • skewX(): sesgado en vertical
  • skewY(): sesgado en horizontal
  • matrix(): combina varios

El método matrix ()

El método matrix() combina todos los métodos 2D de transformación en una sola.

El método de matrix () toma seis parámetros, que contiene funciones matemáticas, que le permite rotar, escala, movimiento (traducir), y el sesgado elementos.

Los parámetros son los siguientes: matriz (scaleX (), skewY (), skewX (), scaleY (), translateX (), translateY ()), no se utilizan las unidades si no parametros.
