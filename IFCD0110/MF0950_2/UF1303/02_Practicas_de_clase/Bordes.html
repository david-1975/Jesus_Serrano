Estilo de borde

La propiedad border-style especifica qué tipo de frontera que se vea.

Se permiten los siguientes valores:

  • dotted - Define un borde de puntosborder-style: dotted
  • dashed - Define un borde punteadoborder-style: dashed
  • solid - Define un borde sólidoborder-style: solid
  • double - Define un borde dobleborder-style: double
  • groove- Define un borde acanalado 3D. El efecto depende del valor de border-colorborder-style: groove
  • ridge- Define un borde estriado 3D. El efecto depende del valor de border-colorborder-style: ridge
  • inset- Define un borde inserción 3D. El efecto depende del valor de border-colorborder-style: inset
  • outset- Define un borde principio 3D. El efecto depende del valor de border-colorborder-style: outset
  • none - Elimina bordeborder-style: No
  • hidden - Define un borde ocultoborder-style: hidden
  • mixed - Define un borde mexcladoborder-style: mixed

Nota: La propiedad border-style puede tener de uno a cuatro valores (por el borde superior, borde derecho, borde inferior y el borde izquierdo).

Ancho del borde

La propiedad border-width especifica la anchura de los cuatro bordes.

La anchura se puede establecer como un tamaño específico (en px, pt, cm, em, etc) o mediante el uso de uno de los tres valores predefinidos: thin (fino), medium (medio), or thick (grueso).

La propiedad border-width puede tener de uno a cuatro valores (por el borde superior, borde derecho, borde inferior y el borde izquierdo).

border-style: solid; border-width: 5px

border-style: solid; border-width: medium

border-style: dotted; border-width: 2px

border-style: dotted; border-width: thick

border-style: double; border-width: 15px

border-style: double; border-width: thick

border-style: solid; border-width: 2px 10px 4px 20px

Color del borde

La propiedad border-color se utiliza para establecer el color de los cuatro bordes.

El color se puede ajustar por:

  • Nombre - indique un nombre de color, como "rojo"
  • Hex - especificar un valor hexadecimal, como "# FF0000"
  • RGB - especificar un valor RGB, así como "rgb (255,0,0)"
  • transparente

La propiedad border-color puede tener de uno a cuatro valores (por el borde superior, borde derecho, borde inferior y el borde izquierdo). 

Si border-color no se establece, se hereda el color del elemento.

border-color: red

border-color: green

border-color: red green blue yellow

Border - Lados individuales

De los ejemplos anteriores se han visto que es posible especificar un borde diferente para cada lado.

En CSS, también hay propiedades para especificar cada uno de los bordes (superior, derecho, inferior e izquierda):

border-top-style: dotted; border-right-style: solid; border-bottom-style: dotted; border-left-style: solid;

El ejemplo anterior muestra el mismo resultado que esto:

border-style: dotted solid;

La propiedad border-style se puede poner con varios valores

  • 4 valores: Se refiere en este orden a arriba, derecha, abajo e izquierda.
  • 3 valores: Se refiere en este orden a arriba, a los lados y abajo.
  • 2 valores: Se refiere en este orden a arriba-abajo y derecha-izquierda.
  • 1 valor: Se refiere a todos los lados.

Border - Abreviación de las propiedades

Como se puede ver en los ejemplos anteriores, hay muchas propiedades a tener en cuenta cuando se trata de bordes.

Para acortar el código, también es posible especificar todas las propiedades de borde individuales en una sola propiedad.

La propiedad border es una propiedad abreviada para las siguientes propiedades de borde individuales:

  • border-width
  • border-style (necesario)
  • border-color

border: 5px solid red;

También puede especificar todas las propiedades de borde individuales para un solo lado:

border-left: 6px solid red; background-color: lightgrey;

Bordes redondeados

La propiedad border-radius se utiliza para añadir bordes redondeados a un elemento, al igual que en border-style se le puede añadir en varios valores:

4 Valores: border-radius:15px 50px 30px 5px;
3 Valores: border-radius:15px 50px 30px;
2 Valores: border-radius:15px 50px;
Valor único: border-radius:100px;

border-radius Establece el borde de las 4 esquinas
border-style Establece el estilo de los 4 lados
border-width Establece la anchura de los 4 bordes

Outline CSS

Outline es una línea que se dibuja alrededor de los elementos, fuera de los bordes, para hacer que el elemento "destaque".

Outline
Border
Contenido

CSS tiene las siguientes propiedades del contorno:

  • outline-style
  • outline-color
  • outline-width
  • outline-offset
  • outline

Nota:  Outline difiere de los border! A diferencia del border, el contorno se dibuja fuera del borde del elemento, y puede superponerse otros contenidos. Además, el outline no es una parte de las dimensiones del elemento; total de anchura y la altura del elemento no se ve afectada por el ancho del contorno.
