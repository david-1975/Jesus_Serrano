Dónde se inserta JavaScript

  • En HTML, el código JavaScript en HTML debe ser insertado entre las etiquetas <script> y </script>.
  • Los scripts pueden ser colocados en el <body>, o en la <head> sección de una página HTML, o en ambos.
  • Los scripts también pueden ser colocados en archivos externos con js/funciones.js
  • El código JavaScript puede ser insertado en línea, en las etiquetas HTML dentro del atributo de evento onClick
Ejemplos de edad de JavaScript pueden utilizar un atributo de tipo: <script type = "text / javascript">. 
No se requiere el atributo type. JavaScript es el lenguaje de scripts por defecto en HTML.

Ejemplo: JavaScript en el Head

<head>
<script>
function myFunction() {
  document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</head>

Ejemplo: JavaScript en el Body

<body> 

<h1>A Web Page</h1>
<p id="demo">A Paragraph</p>
<button type="button" onclick="myFunction()">Try it</button>

<script>
function myFunction() {
 document.getElementById("demo").innerHTML = "Paragraph changed.";
}

</body>
