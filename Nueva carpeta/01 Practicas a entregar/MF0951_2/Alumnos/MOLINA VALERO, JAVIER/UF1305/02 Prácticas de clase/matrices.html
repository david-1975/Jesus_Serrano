splice()

El método splice() se puede utilizar para añadir nuevos elementos a una matriz.

El primer parámetro (1,..) define la posición donde se deben nuevos elementos añadidos (empalmados in).

El resto de los parámetros (0,0,........ ) definen los nuevos elementos que se añaden .

El splice()método devuelve una matriz con los elementos eliminados

El segundo parámetro (0,0) define el número de elementos debe ser eliminado .

 array.splice(2, 0, "Lemon", "Kiwi")
slice()

El slice()método rebana un pedazo de una matriz en una matriz nueva.

El slice()método puede tomar dos argumentos como slice(1, 3).

El método selecciona entonces elementos de la argumento comienzo, y hasta (pero no incluyendo) el argumento final.

Si se omite el argumento final, el slice() método rebana el resto de la matriz.

 array.slice()
sort()

El método  sort()ordena una matriz alfabéticamente

Por defecto, la sort()función ordena los valores como cadenas .

Sin embargo, si los números se clasifican como cadenas, "25" es más grande que el "100", ya que "2" es más grande que el "1".

Debido a esto, el sort()método producirá resultado incorrecto al ordenar los números.

Puedes solucionar este problema proporcionando una función de comparación 

array.sort()

array.sort(function(a, b){return a - b});
filter()

El método filter () crea una matriz llena de todos los elementos de la matriz que pasan una prueba (proporcionado como una función).

Nota: filter () no se ejecuta la función de elementos de la matriz sin valores.

Nota: filter () no cambia la matriz original.

 array.filter(function(currentValue, index, arr), thisValue)
