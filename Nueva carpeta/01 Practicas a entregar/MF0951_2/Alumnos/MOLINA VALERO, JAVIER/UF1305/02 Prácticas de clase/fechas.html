Métodos fechas (Date)

El concepto de fecha

Por defecto, JavaScript utilizará la zona horaria del navegador y mostrar una fecha como una cadena de texto completo:

Mar Mar 19 de 2019 12:50:07 GMT + 0100 (Hora Estándar de Europa central)

Los objetos fecha se crean con el new Date() constructor.

new Date()crea un nuevo objeto Date con la fecha y hora actual

Métodos numéricos

Método Descripción Síntasix  
new Date() Crea un nuevo objeto de tipo Date con la fecha y hora actual new Date()
getFullYear() Coge el año de una fecha como número con los dígitos the year as a four digit number(yyyy) Date.getFullYear()
getMonth() El método getMonth() devuelve el mes de una fecha como un número (0-11): getMonth()
getDate() El método getDate() devuelve el día de una fecha como un número (1-31) getDate()
getHours() El método getHours() devuelve las horas de una fecha como un número (0-23) getHours()
getMinutes() El método getMinutes() devuelve los minutos de una fecha como un número (0-59) getMinutes()
getSeconds() El método getSeconds() devuelve los segundos de una fecha como un número (0-59) getSeconds()
getMilliseconds() El método getMilliseconds() devuelve los milisegundos de una fecha como un número (0-999) getMilliseconds()
getTime() Get the time (milliseconds since January 1, 1970) getTime()
getDay() El método getDay() devuelve el día de la semana de una fecha como un número (0-6) empezando por domingo getDay()
