Las posibilidades de visualización de JavaScript

JavaScript puede alterar los datos "Display" de diferentes maneras:

  • La escritura en un elemento HTML, usando innerHTML.
  • La escritura en la salida HTML utilizando document.write().
  • Escribir en un cuadro de alerta, utilizando window.alert().
  • Escribir en la consola del navegador, usando console.log().

Usando innerHTML

Para acceder al contenido HTMl de un elemento se utiliza innerHTML

Usando document.write

Para propósitos de prueba, es conveniente utilizar document.write()

Usando window.alert

Se puede utilizar un cuadro de alerta para mostrar los datos

Usando console.log

Para propósitos de depuración, se puede utilizar el método console.log() para mostrar los datos.

confirm()

El método confirm() muestra un cuadro de diálogo con un mensaje especificado, junto con un clic en Aceptar y un botón Cancelar.

confirm(message)

alert()

El método de alert() muestra un cuadro de alerta con un mensaje especificado y un botón Aceptar.

alert(message)

prompt()

El método prompt() devuelve el valor de entrada si el usuario hace clic en "OK". Si el usuario hace clic en "cancelar" el método devuelve null.

prompt(text, textoPorDefecto)

open()

El método open() abre una nueva ventana del navegador.

Consejo: Utilice el método close() para cerrar la ventana.

window.open(URL, name, specs, replace)

close()

El método close() cierra la ventana actual.

Consejo: Este método se utiliza a menudo junto con el método open().

window.close()

print()

El método print() imprime el contenido de la ventana actual.

El método print() abre el cuadro de diálogo Imprimir, que permite al usuario seleccionar las opciones de impresión preferidos.

window.print()

setInterval()

El método setInterval () llama a una función o evalúa una expresión a intervalos especificados (en milisegundos).

El método setInterval () continuará llamando a la función hasta clearInterval () se llama, o la ventana está cerrada.

setInterval(function, milliseconds, param1, param2, ...)

setTimeout()

El método setTimeout() llama a una función o evalúa una expresión después de un número especificado de milisegundos.

setTimeout(function, milliseconds, param1, param2, ...)

btoa()

El método btoa () codifica una cadena en base-64.

Este método utiliza el "AZ", "az", "0-9", y "=" caracteres "+" "/" para codificar la cadena.
