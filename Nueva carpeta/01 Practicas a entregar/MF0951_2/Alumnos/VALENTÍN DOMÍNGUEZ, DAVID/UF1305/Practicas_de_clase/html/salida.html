JavaScript salida

Las posibilidades de visualización de JavaScript

JavaScript puede alterar los datos "Display" de diferentes maneras:

  • La escritura en un elemento HTML, usando  innerHTML.
  • La escritura en la salida HTML utilizando document.write().
  • Escribir en un cuadro de alerta, utilizando  window.alert().
  • Escribir en la consola del navegador, usando  console.log().

 

Usando innerHtml

Para acceder al contenido HTML de un elemento se utiliza innerHTML.

Usando document.write

Para propósitos de prueba se utiliza document.write() para mostrar los datos.

Usando window.alert

Para propósitos informativo utilizar el método window.alert() para mostrar los datos.

Usando console.log

Para propósitos de depuración utilizar el método console.log() para mostrar los datos. (F12)

confirm()

El método de confirmación () muestra un cuadro de diálogo con un mensaje especificado, junto con un clic en Aceptar y un botón Cancelar.

confirm(message)

alert()

El método de alerta () muestra un cuadro de alerta con un mensaje especificado y un botón Aceptar.

alert(message)

prompt()

El método prompt () muestra un cuadro de diálogo que solicita al visitante una entrada de datos

prompt(text, defaultText)

window.open

El método open() abre una nueva ventana del navegador.

window.open(URL, name, specs, replace)

window.close

El método close() muestra un cuadro de diálogo de alerta con contendio opcional especificado y un botón Aceptar

window.close()

window.print;

El método print() muestra un cuadro de diálogo que solicita una entrada de datos, junto a un botón Aceptar y Cancelar

window.print();

Nota:Cuando un cuadro de mensaje aparece el usuario no podrá acceder a otras partes de la página hasta que la caja esté cerrada.

setInterval

El método setInterval()llama a una función o evalúa una expresión a intervalos especificados (en milisegundos). continuará llamando a la función hasta que clearInterval (), o la ventana está cerrada. El valor ID devuelto por setInterval () se utiliza como el parámetro para el método clearInterval ().

setInterval(function, milliseconds, param1, param2, ...)

setTimeout

El método setTimeout()El método setTimeout () llama a una función o evalúa una expresión después de un número especificado de milisegundos.

setTimeout(function, milliseconds, param1, param2, ...)

btoa() codifica

atob() descodifica

El método btoa ()codifica una cadena en base-64. Este método utiliza el "AZ", "az", "0-9", y "=" caracteres "+" "/" para codificar la cadena. Consejo: Utilice la atob () método para decodificar una cadena de base-64 codificada.

Nota: el método atob () no es compatible con IE9 y versiones anteriores.

window.atob(encodedStr)
