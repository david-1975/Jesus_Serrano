Métodos numéricos

Método Descripción Síntasix  
new Array() Crea un nuevo objeto de tipo Array con los datos existentes en variables new Date()
split() Se utiliza para dividir una cadena en una matriz de subcadenas, y devuelve la nueva matriz. string.split(separator, limit)
concat() Une dos o más matrices y devuelve una copia de las matrices unidas array1.concat(array2,)
copyWithin() Copia elementos de matriz dentro de la matriz, hacia y desde las posiciones especificadas array.copyWithin(target, start, end)
entries() Devuelve un par clave / valor Array Iteration Object array.entries()
toString() Convierte una matriz a una cadena de (separados por comas) valores de la matriz. array.toString()
join() Se une a todos los elementos de la matriz en una cadena. array.join(separator)
pop() Elimina el último elemento de una matriz array.pop()
push() Añade un nuevo elemento a una matriz (al final) array.push(item1, item2, ..., itemX)
shift() Elimina el primer elemento de matriz y "desplaza" todos los otros elementos a un índice inferior. array.shift()
unshift() Añade un nuevo elemento a una matriz (al principio) array.unshift(item1, item2, ..., itemX)
length() Ofrece una forma fácil de añadir un nuevo elemento a una matriz array.length
splice()  Se puede utilizar para añadir nuevos elementos a una matriz array.splice(index, howmany, item1)
delete() Los elementos se pueden eliminar mediante el operador de JavaScript delete()
slice() Rebana un pedazo de una matriz en una matriz nueva. array.slice(start, end)
sort() Método ordena una matriz alfabéticamente array.sort(compareFunction)
reverse() Método invierte los elementos de una matriz. array.reverse()
entries()   array.entries()
every()   array.every(function(currentValue, index, arr), thisValue)
fill()   array.fill(value, start, end)
filter()   array.filter(function(currentValue, index, arr), thisValue)
find()   array.find(function(currentValue, index, arr),thisValue)
findIndex()   array.findIndex(function(currentValue, index, arr), thisValue)
forEach()   array.forEach(function(currentValue, index, arr), thisValue)
from() Array.from(object, mapFunction, thisValue)
includes() array.includes(element, start)
indexOf() array.indexOf(item, start)
isArray() Array.isArray(obj)
keys() array.keys()
lastIndexOf() array.lastIndexOf(item, start)
map() array.map(function(currentValue, index, arr), thisValue)
reduce() array.reduce(function(total, currentValue, currentIndex, arr), initialValue)
reduceRight() array.reduceRight(function(total, currentValue, currentIndex, arr), initialValue)
some() array.some(function(currentValue, index, arr), thisValue)
valueOf() array.valueOf()
