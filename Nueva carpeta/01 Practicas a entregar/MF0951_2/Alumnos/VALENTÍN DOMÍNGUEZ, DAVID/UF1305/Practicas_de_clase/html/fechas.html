Métodos Fechas (Date)

El concepto de Fecha

Por defecto, JavaScript utilizará la zona horaria del navegador y mostrar una fecha como una cadena de texto completo:

Mar Mar 19 de 2019 12:50:06 GMT + 0100 (Hora Estándar de Europa central)

Métodos numéricos

Método Descripción Síntasix  
new Date() crea un nuevo objeto Date con la fecha y hora actual new Date()
getFullYear() Obtener el año como un número de cuatro dígitos (aaaa) Date.getFullYear()
getMonth() Obtener el mes como un número (0-11) Date.getMonth()
getDate() Obtener el día como un número (1-31) Date.getDate()
getHours() Consigue la hora (0-23) Date.getHours()
getMinutes() Consigue el minuto (0-59) Date.getMinutes()
getSeconds() Consigue el segundo (0-59) Date.getSeconds()
getMilliseconds() Obtener el milisegundo (0-999) Date.getMilliseconds()
getTime() Obtener el tiempo (milisegundos desde el 1 de enero de 1970) Date.getTime()
getDay() Obtener el día de la semana como un número (0-6) Date.getDay()
