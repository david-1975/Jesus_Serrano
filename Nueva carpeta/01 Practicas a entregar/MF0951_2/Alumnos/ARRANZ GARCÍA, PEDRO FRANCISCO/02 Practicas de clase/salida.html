Las posibilidades de visualización de JS

JavaScript can "display" data in different ways:

  • Writing into an HTML element, using innerHTML.
  • Writing into the HTML output using document.write().
  • Writing into an alert box, using window.alert().
  • Writing into the browser console, using console.log()

Las posibilidades de visualización de JS

JavaScript can "display" data in different ways:

  • Writing into an HTML element, using innerHTML.
  • Writing into the HTML output using document.write().
  • Writing into an alert box, using window.alert().
  • Writing into the browser console, using console.log()

Ejemplo de confirm()

The confirm() method displays a dialog box with a specified message, along with an OK and a Cancel button.

confirm(message)

Ejemplo de alert()

The alert() method displays an alert box with a specified message and an OK button. See the expample below.

alert(message)

Ejemplo de prompt()

The prompt() method displays a dialog box that prompts the visitor for input. That input can then be stored.

prompt(text, defaultText)

Ejemplo de open()

The open() method opens a new browser window. Use the close() method to close the window.

open("https://www.hola.com");

Ejemplos de métodos...

Ejemplo de close()

Use open() to open a new window, and close() to close the new window.

myWindow.close()

Ejemplo de print()

El método print() imprime la página. The print() method opens the Print Dialog Box.

window.print()

resizeTo() y moveTo()

resizeTo() sets a new width and height. moveTo la desplaza.

myWindow2.resizeTo(x,y);

myWindow2.moveTo(x,y);

SetInterval() y SetTimeout

setInterval(function(){ alert("Hello"); }, 3000);

setTimeout(function(){ alert("Hello"); }, 3000);
