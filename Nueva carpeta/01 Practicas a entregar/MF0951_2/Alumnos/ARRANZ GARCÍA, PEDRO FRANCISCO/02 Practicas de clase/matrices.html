Métodos de array

Método Descripción Síntasix  
new Array() Crea un nuevo objeto de tipo matriz con los datos existentes en una variable. new Array()
join() Joins all array elements into a string but in addition you can specify the separator. matriz1.join(" * ")
pop() Removes the last element from an array. matriz1.pop()
push() Adds an element to an array. matriz1.push("Kia")
shift() Removes the first element and "shifts" other elements to lower index. matriz1.shift()
splice() Elimina elementos y añade otros nuevos al array. matriz1.splice(2, 0, "Kia", "Honda")
concat() Crea un array mediante la unión de dos dos arrays ya existentes matriz1.concat(matriz2)
sort() The sort() method sorts an array alphabetically matriz1.sort()
