La etiqueta <script>

En HTML, el código JavaScript se puede insertar entre etiquetas <script> y </script>.

Se pueden poner cualquier número de scripts en un documento HTML.

Los scripts se pueden poner en el <body>, o en el <head> o en ambos.

Poniendo los scripts al final del <body> mejora la velocidad de display, porque la compilación del script ralentiza el display.

Los scripts también se pueden poner en ficheros externos con extensión .js. Poner el nombre del fichero script en el atributo "src" de una etiqueta <script>, dela siguiente forma:<script src="myScript.js"> </script>

Se puede poner la referencia a script externos tanto en el <head> como en el <body>.

Los scripts externos no pueden contener etiquteas <script>.

El código Javascript se puede insertar en línea, en las etiquetas HTML, dentro del atributo de evento, por ejemplo en el onClick

Ejemplo de JS en el head

<!DOCTYPE html>
<html>

<head>
<script>
function myFunction() {
document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</head>
<body>

<h1>A Web Page</h1>
<p id="demo">A Paragraph</p>
<button type="button" onclick="myFunction()">Try it</button>

</body>
</html>

Posibilidades de Display de JavaScript

JavaScript puede sacar datos de diferentes formas:

  • Escribiendo en un elemento HTML, usando innerHTML.
  • Escribiendo en el output de HTML usando document.write().
  • Escribiendo en una caja de alerta, usando window.alert().
  • Escribiendo en la consola del navegador, usando console.log().

Usando innerHTML

Para acceder al elemento HTML, JavaScript puede utilizar el método document.getElementById(id)

El atributo id define el elemento HTML. La propiedad innerHTML define el contenido HTML:

<!DOCTYPE html>
<html>
<body>

<h1>My First Web Page</h1>
<p>My First Paragraph</p>

<p id="demo"></p>

<script>
document.getElementById("demo").innerHTML = 5 + 6;
</script>

</body>
</html>
