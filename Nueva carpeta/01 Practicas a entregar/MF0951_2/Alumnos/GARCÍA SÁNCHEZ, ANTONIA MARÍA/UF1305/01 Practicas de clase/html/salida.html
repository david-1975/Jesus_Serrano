Posibilidades de visualización de JavaScript

JavaScript puede sacar datos de diferentes formas:

  • Escribiendo en un elemento HTML, usando innerHTML.
  • Escribiendo en el output de HTML usando document.write().
  • Escribiendo en una caja de alerta, usando window.alert().
  • Escribiendo en la consola del navegador, usando console.log().

Cuadros de diálogos de ventana

JavaScript puede mostar cuadros de diálogos a través del elemento window del DOM:


confirm()

El método confirm() muestra una caja de diálogo con un mensaje especifico, con un botón OK y Cancel. Se utiliza a menudo para que el usuario verifique algo.

Devuelve true si el usuario pulsa "OK", en caso contrario devuelve false.

confirm(mensaje)

alert()

El método alert() muestra una caja de diálogo con un mensaje especifico y un botón de OK.

Se usa para asegurar que el usuario está informado de algo

alert(mensaje)

prompt()

El método prompt() muestra una caja de diálogo al usuario para que incorpore una entrada de datos.

Devuelve el valor de la entrada si el usuario pulsa "OK". Si el usuario pulsa "cancel" devuelve null.

prompt(texto, defaultText)

open()

El método open(), abre una nueva ventana de navegador.

window.open(URL, name, specs, replace)

close()

El método close(), cierra una nueva ventana de navegador.

window.close()

print()

El método printe(), imprime el contenido de la ventana.

window.print()

Método Uso Sintaxis Ejemplo
open() Abre una nueva ventana. window.open()
blur() Quita el foco de la ventana actual window.blur()
focus() Pone el foco en la ventana actual window.focus()
moveBy() Mueve una ventana un nº de pixels, relativo a las coordenadas actuales. window.moveBy(x, y)
moveTo() Mueve la ventana a las coordenadas que se especifiquen. window.moveTo(x, y)
resizeBy() Cambia el tamaño de la ventana una cantidad relativa a su tamaño actual. resizeBy(width, height)
resizeTo() Cambia el tamaño al ancho y alto especificado window.resizeTo(width, height)
setInterval() Llama a una función o evalúa una expresión un tiempo determinado (en milisegundos) y lo repite continuamente. setInterval(function, milliseconds, param1, param2, ...)
clearInterval() Para un temporizador iniciado con setInterval. clearInterval(var)
setTimeout Llama a una función o evalúa una expresión despues de un tiempo determinado (en milisegundos). setTimeout(function, milliseconds, param1, param2, ...)
clearTimeout() Para un temporizador iniciado con setTimeout. clearTimeout(var)
btoa() Para codificar a un string. window.btoa(str)
atob() Para decodificar un texto codificado. window.atob(str)
