Métodos Matrices (Array)

El concepto de Array

Por defecto, JavaScript utiliza la zona horaria del navegador y muestra la fecha como una cadena de texto:

Tue Mar 19 2019 12:50:07 GMT+0100 (hora estándar de Europa central)

Los objetos de fecha se crean con new Date(), que crean una fecha con la fecha y hora actual.

Los métodos numéricos devuelven un valor nuevo y no cambian la variable original, y puede utilizarse con cualquier tipo de número (literales, variables o expresiones).

Métodos de fechas

Método Descripción Síntasix  
new Array() Crea una nueva matriz. new array())
concat() Concatena dos o más arrays y devuelve una copia de los arrays unidos. array1.concat(array2, array3, ..., arrayX)
toString() Convierte un array en cadena. array.toString()
join() Convierte un array en cadena, y se le puede indicar el tipo de separador. array.join(separator)
push() Añade nuevos elementos al final de un array y devuelve la longitud. array.push(item1, item2, ..., itemX)
unshift() Añade nuevos elementos al principio de un array y devuelve la longitud. array.unshift(item1, item2, ..., itemX)
splice() Añade/elimina elementos a/de un array y devuelve los elementos añadidos/eliminados. Se indica la posición desde donde añadir/eliminar, cuantos añadir/eliminar (0 para no eliminar ninguno), y a continuación los itemas a añadir. array.splice(index, howmany, item1, ....., itemX)
pop() Elimina el último elemento de un array y devuelve ese elemento. array.pop
shift() Elimina el primer elemento de un array y devuelve ese elemento. array.shift
slice() Devuelve los elementos seleccionados del array en un nuevo array (en una copia). Desde la posición inicial a la final (final no incluido). array.slice(start, end)
delete Borra un item de un array y deja el hueco. delete array[posicion]
sort() Ordena alfabéticamente (orden ascendente)los items de un array . Cuando se quiere ordenar numéricamente añadir una función de comparación como por ej: "function(a, b){return a-b}" para orden ascendente, "function(a, b){return b-a}" para orden descendente, "function(a, b){return 0.5 - Math.random()}" para orden aleatorio. array.sort(compareFunction)]
reverse() Ordena alfabéticamente (orden descendente) los items de un array. array.reverse()
copyWithin() Copia elementos dentro de un array desde una posicion inicio a una final. array.copyWithin(target, start, end)
fill() Rellena los elementos de un array con un valor estático. array.fill(value, start, end)
every() Comprueba si todos los elementos de un array pasan un test (de una función). Ejecuta la función para cada elemento del array. Devuelve true o false. array.every(function(currentValue, index, arr), thisValue)
filter() Crea un nuevo array con los elementos que cumplen una condición. array.filter(function(currentValue, index, arr), thisValue)
find() Devuelve el valor del primer elemento que cumple una condición. array.find(function(currentValue, index, arr), thisValue)
findIndex() Devuelve el índice del primer elemento cuyo valor cumple una condición. array.findIndex(function(currentValue, index, arr), thisValue)
from() Crea un array desde cualquier objeto con propiedad length o iterable. array.from(object, mapFunction, thisValue)
includes() Busca si un elemento está incluido en el array. Devuelve true o false. array.includes(element, start)
indexOf() Busca un elemento en el array y devuelve su posición. array.indexOf(element, start)
lastIndexOf() Busca un elemento en el array desde el final y devuelve su posición. array.lastIndexOf(element, start)
isArray() Determina si un objeto es un array. Devuelve true o false. array.isArray(obj)
forEach() Llama a una función para cada elemento. array.forEach(function(currentValue, index, arr), thisValue)

 

Method Description

entries() Returns a key/value pair Array Iteration Object

fill() Fill the elements in an array with a static value
filter() Creates a new array with every element in an array that pass a test
find() Returns the value of the first element in an array that pass a test
findIndex() Returns the index of the first element in an array that pass a test
forEach() Calls a function for each array element
from() Creates an array from an object
includes() Check if an array contains the specified element
indexOf() Search the array for an element and returns its position
isArray() Checks whether an object is an array
join() Joins all elements of an array into a string
keys() Returns a Array Iteration Object, containing the keys of the original array
lastIndexOf() Search the array for an element, starting at the end, and returns its position
map() Creates a new array with the result of calling a function for each array element
pop() Removes the last element of an array, and returns that element
push() Adds new elements to the end of an array, and returns the new length
reduce() Reduce the values of an array to a single value (going left-to-right)
reduceRight() Reduce the values of an array to a single value (going right-to-left)

toString() Converts an array to a string, and returns the result
