¿Dónde puede ser insertado JavaScript?

  • El código JavaScript en HTML debe ser insertado entre las etiquetas <script> y </script>.
  • Los scripts pueden ser colocados en la sección <body>, o en el <head> de una página HTML, o en ambos.
  • Los scripts también pueden ser colocados en archivos externos como js/funciones.js
  • El código JavaScript puede ser insertado en línea, en las etiquetas HTML, en el atributo de evento onclick.

Ejemplo de código JavaScript en la sección Head

<head>
<script>
 function miFuncion1() {
document.getElementById("demo1").innerHTML = "Párrafo Cambiado.";
document.getElementById('demo1').style.color = "blue"; }
 </script>
</head>

<body>
<p id="demo1">Este párarrafo va a ser cambiado.</p>
<button type="button" onClick="miFuncion1()">Pulsar</button>
</body>

Ejemplo de uso

Cuando se hace clic en el botón se invoca (llama) a la función miFuncion1() situada en la sección <head> entre etiquetas <script>.

Este párarrafo va a ser cambiado.

Ejemplo de código JavaScript en la sección Body

Ejemplo: JavaScript en un documento externo JS

<head>
<script src="js/funciones.js">
</head>

<body>

<p id="demo3">Este párarrafo va a ser cambiado.</p>
<button type="button" onClick="miFuncion3()">Pulsar</button>
</body>

Ejemplo de uso

Cuando se hace clic en el botón se invoca (llama) a la función miFuncion3() situada en un documento externo de JavaScript.

Este párarrafo va a ser cambiado.

Ejemplo: JavaScript en una etiqueta HTML

<body>

<p id="demo4">Este párarrafo va a ser cambiado.</p>

<button type="button" onClick="
document.getElementById('demo4').innerHTML = 'Párrafo cambiado.';
document.getElementById('demo4').style.color = 'blue';
">Pulsar</button>
</body>

Ejemplo de uso

Cuando se hace clic en el botón se ejecuta la sentencia de JavaScript escrita dentro del atributo onClick d ela etiqueta HTML.

Este párarrafo va a ser cambiado.
