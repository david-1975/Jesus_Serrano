Métodos string (cadenas)

El concepto de cadenas

Un string (cadena), es cero o más caracteres escritos entre comillas. Se utiliza como valor en propiedades de objetos, matrices y variables.

Las cadenas JavaScript se utilizan para almacenar y manipular texto. Los métodos son funciones que permiten manipular las cadenas de texto.

Secuencias de escape JavaScript
Código Resultado
\b Espacio en blanco
\f Avance de página
\n Nueva línea
\r Retorno de carro
\t Tabulador Horizontal

La secuencia \n inserta una línea nueva en una cadena

Secuencias de escape JavaScript
Código Resultado Descripción
\' ' Comillas simples
\" " Comillas dobles
\\ \ Barra invertida
\u00F1 ñ Letra ñ minúscula
\u00D1 Ñ Letra ñ mayúscula

La secuencia \" inserta una comilla doble en una cadena

Método Descripción  
length Devuelve la longitud de una cadena.
charAt ( ) Devuelve el carácter en el índice especificado (posición).
charCodeAt ( ) Devuelve el índice del caracter Unicode especificado.
fromCharCode ( ) Convierte los valores Unicode en caracteres.
toUpperCase ( ) Convierte una cadena en mayúsculas.
toLowerCase ( ) Convierte una cadena a minúsculas.
concat ( ) Convierte dos o más cadenas en una nueva cadena concatenada.
endsWith ( ) Comprueba si una cadena termina con una cadena/caracteres especificados.
includes ( ) Comprueba si una cadena contiene la cadena/caracteres especificados.
indexOf ( ) Devuelve la posición de la primera aparición encontrada de un valor especificado en una cadena.
lastIndexOf ( ) Devuelve la posición de la última aparición encontrada de un valor especificado en una cadena.
repeat ( ) Devuelve una cadena nueva con un número especificado de copias de una cadena existente
replace( ) Busca una cadena con una expresión regular y devuelve una nueva cadena sustituyendo los valores especificados.
search ( ) Busca una cadena para un valor especificado o una expresión regular y devuelve la posición de la coincidencia
slice ( ) Extrae parte de una cadena y devuelve una nueva cadena con la parte extraída.
split ( ) Divide una cadena en una matriz de subcadenas
startsWith ( ) Comprueba si una cadena empieza con una cadena/caracteres especificados.
substr ( ) Extrae partes de una cadena, comenzando en una posición especificada y la longitud de caracteres a mostrar.
substring ( ) Extrae los caracteres de una cadena, entre dos índices especificados (posición).
trim() Elimina el espacio en blanco de ambos extremos de una cadena
match ( ) Busca una cadena que coincida con una expresión regular y devuelve las coincidencias separadas por comas
toString( ) Convierte un objeto en una cadena de caracteres.
