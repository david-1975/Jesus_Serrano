JavaScript salida

Las posibilidades de visualización de JavaScript

JavaScript puede alterar los datos "Display" de diferentes maneras:

  • La escritura en un elemento HTML, usando  innerHTML.
  • La escritura en la salida HTML utilizando document.write().
  • Escribir en un cuadro de alerta, utilizando  window.alert().
  • Escribir en la consola del navegador, usando  console.log().

 

Usando innerHtml

Para acceder al contenido HTML de un elemento se utiliza innerHTML. Podemos cambiar el contenido de cualquier elemento: texto para <p> , imagen (src) para <img> ...

Usando document.write

Para propósitos de prueba se utiliza document.write(). Modifica el contenido de todo el documento.

Usando window.alert

Para propósitos informativo utilizar el método window.alert() para mostrar los datos en una ventana de alerta.

Usando console.log

Para propósitos de depuración utilizar el método console.log() para mostrar los datos. En F12/console

confirm()

El método de confirmación () muestra un cuadro de diálogo con un mensaje especificado, junto con un clic en Aceptar y un botón Cancelar.

confirm(message)

alert()

El método de alerta () muestra un cuadro de alerta con un mensaje especificado y un botón Aceptar.

alert(message)

prompt()

El método prompt () muestra un cuadro de diálogo que solicita al visitante una entrada de datos

prompt(text, defaultText)


open()

El método open () abre una nueva ventana del navegador.

window.open("ruta")

close()

El método close () cierra la ventana actual.

window.close()

print()

El método print () imprime el contenido de la ventana actual. Abre el cuadro de diálofo Imprimir para seleccionar opciones

window.print()


setInterval()

Llama a una función o evalúa una expresión a intervalos especificados (en milisegundos).Repite

setInterval(function, milliseconds, param1, param2, ...)

setTimeout()

Llama a una función o evalúa una expresión después de los milisegundos especificados. Una vez

setTimeout(function, milliseconds, param1, param2, ...)

atob()y btoa()

Codifica (btoa) o decodifica (atob) una cadena en base 64.

window.atob(var)

window.btoa(var)
