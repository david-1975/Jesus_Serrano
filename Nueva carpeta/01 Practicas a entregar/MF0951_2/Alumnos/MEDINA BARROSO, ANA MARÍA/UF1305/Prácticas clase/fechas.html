Métodos de fecha Date()

Creación de objetos de fecha

De forma predeterminada, JavaScript utilizará la zona horaria del navegador y mostrará una fecha como una cadena de texto completa: Mar 19 de marzo 2019 12:50:33 GMT + 0100 (hora estándar de Europa central)

Los objetos de fecha se crean con el new Date()

Métodos de fecha

Método Descripción Síntasix  
new Date() crea un nuevo objeto de fecha con la fecha y hora actuales. new Date()
getFullYear() Devuelve el año de una fecha como un número de cuatro dígitos: variable.getFullYear()
getMonth() :() Devuelve el mes de una fecha como un número (0-11) variable.getMonth();
getDate() Devuelve el día de una fecha como un número (1-31). variable.getDate()
getHours() Devuelve las horas de una fecha como un número (0-23): variable.getHours()
getMinutes() Devuelve los minutos de una fecha como un número (0-59): variable.getMinutes()
getSeconds() Devuelve los segundos de una fecha como un número (0-59): variable.getSeconds()
setTime() Convierte un número en una cadena y lo representa en una base aritmética. number.toString(radix)
