Donde se inserta Javascript

 

  • INTERNO: (En el archivo HTML). El código Javascript debe ser insertado entre <script> y </script> Los scripts se pueden colocar en <body>, o en la sección <head> de la página HTML, o en ambos.
    <script>
    function myFunction() {
     document.getElementById("demo").innerHTML = "Paragraph changed.";
    }
    </script>
  • EXTERNO: (En archivos .js) Los scripts tambien pueden ser colocados en archivos externos. Ej: miarchivo.js
    function miFuncion() {
    document.getElementById("demostracion").innerHTML = "Párrafo cambiado.";
    }

    Es necesario crear la referencia de este archivo externo, se puede poner tanto en el <head> como en <body> y tantas referencias a archivos como se necesiten.

    <script src="funciones.js"> </script>

    Para poder utilizar una función js del archivo externo solamente hay que indicar el nombre de la función.

    <button type="button" onclick="miFuncion()">Try it</button>
