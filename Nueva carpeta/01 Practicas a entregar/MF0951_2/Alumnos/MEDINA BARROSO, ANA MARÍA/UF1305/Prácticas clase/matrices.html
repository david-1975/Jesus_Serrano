Matrices (Arrays)

Las matrices de JavaScript se utilizan para almacenar varios valores en una sola variable

Los objetos de fecha se crean con el new Date()

Métodos de array

El método
Método Descripción Síntasix  
new Array() Crea un nuevo objeto tipo Array con los datos que indiquemos new Array()
push() Inserta un nuevo elemento al final y devuelve la nueva longitud de la matriz colores.push("Kiwi");
unshift() Inserta un nuevo elemento al principio y devuelve la nueva longitud de la matriz colores.unshift("Kiwi")
pop() Elimina el último elemento y devuelve ese valor. frutas.pop(frutas.length-1)
shift() Elimina el primer elemento de la matriz y desplaza los demás. Devuelve la cadena eliminada: frutas.shift(frutas)
delete() Elimina el contenido de un elemento de la matriz pero la mantiene. delete frutas[posicion];
splice() Inserta o elimina varios elementos a la vez. (desde donde, cuantos, valores a insertar) frutas.splice(2, 0, "Limón", "Kiwi");
concat () Une dos o más matrices en una. var lista= frutas.concat(verduras, otros);
slice() Corta una parte de una matriz en una nueva matriz.  
sort() y reverse() sort() Ordena una matriz alfabéticamente: y reverse() invierte el orden.  
copyWithin() copia elementos de matriz dentro de la matriz, hacia y desde las posiciones especificadas. fruits.copyWithin(2, 0);
