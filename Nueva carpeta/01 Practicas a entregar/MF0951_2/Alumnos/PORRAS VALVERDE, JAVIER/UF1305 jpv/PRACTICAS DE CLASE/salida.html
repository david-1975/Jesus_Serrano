JavaScript salida

Las posibilidades de visualización de JavaScript

JavaScript puede alterar los datos "Display" de diferentes maneras:

  • La escritura en un elemento HTML, usando  innerHTML.
  • La escritura en la salida HTML utilizando document.write().
  • Escribir en un cuadro de alerta, utilizando  window.alert().
  • Escribir en la consola del navegador, usando  console.log().

 

Usando innerHtml

Para acceder al contenido HTML de un elemento se utiliza innerHTML.

Usando document.write

Para propósitos de prueba se utiliza document.write() para mostrar los datos.

Usando window.alert

Para propósitos informativo utilizar el método window.alert() para mostrar los datos.

Usando console.log

Para propósitos de depuración utilizar el método console.log() para mostrar los datos.

confirm

El método de confirmación () muestra un cuadro de diálogo con un mensaje especificado, junto con un clic en Aceptar y un botón Cancelar.

confirm(message)

Alert

El método de alerta () muestra un cuadro de alerta con un mensaje especificado y un botón Aceptar.

alert(message)

Prompt

El método prompt () devuelve el valor de entrada si el usuario hace clic en "OK". Si el usuario hace clic en "cancelar" el método devuelve null.

prompt(text, defaultText)

open

El método abierto () abre un flujo de salida para recoger la salida de cualquier document.write () o document.writeln () métodos.

document.open(MIMEtype, replace())

close

La propiedad cerrada devuelve un valor booleano que indica si una ventana se ha cerrado o no.

window.closed

print

El método de impresión () imprime el contenido de la ventana actual. El método de impresión () abre el cuadro de diálogo Imprimir, que permite al usuario seleccionar las opciones de impresión preferidos.

window.print();

setInterval

El método setInterval () llama a una función o evalúa una expresión a intervalos especificados (en milisegundos). El método setInterval () continuará llamando a la función hasta clearInterval () se llama, o la ventana está cerrada

setInterval(function, milliseconds, param1, param2, ...)

set timeout

El método setTimeout () llama a una función o evalúa una expresión después de un número especificado de milisegundos.

setTimeout(function, milliseconds, param1, param2, ...)

Atob

El método atob () descodifica una cadena de base-64 codificada. Este método descodifica una cadena de datos que han sido codificados por el btoa () método.

window.atob(encodedStr)

btoa

El método btoa () codifica una cadena en base-64. Este método utiliza el "AZ", "az", "0-9", y "=" caracteres "+" "/" para codificar la cadena.

window.btoa(str)

moveBy

El método moveBy () se mueve una ventana de un número especificado de píxeles con respecto a sus coordenadas actuales.

window.moveBy(x, y

moveTo

El método moveTo () mueve el borde izquierdo y la parte superior de una ventana a las coordenadas especificadas.

window.moveTo(x, y)
