Métodos Fecha

Método Descripción Síntasix  
newArray() Crea un nuevo objeto de tipo matriz con los datos existentes en variables. .new array()
toString() Convierte una matriz a una cadena de (separados por comas) valores de la matriz. .toString()
join() Se comporta igual toString(), pero además se puede especificar el separador: .join()
pop() Elimina el último elemento de una matriz: .pop()
push() Añade un nuevo elemento a una matriz (al final): .push()
shift() Elimina el primer elemento de matriz y "desplaza" todos los otros elementos a un índice inferior. .shift()
unshift() Añade un nuevo elemento a una matriz (al principio) .unshift()
length Ofrece una forma fácil de añadir un nuevo elemento a una matriz: .length()
splice() El método de empalme () añade / quita artículos a / desde una matriz, y devuelve el elemento (s) eliminado. .splice()
concat() crea una nueva matriz mediante la fusión (concatenar) matrices existentes: concat()
