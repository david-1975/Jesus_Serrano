JavaScript puede cambiar el contenido HTML

Uno de los muchos métodos JavaScript HTML es getElementById().

En este ejemplo se utiliza el método de "encontrar" un elemento HTML (con id = "demo")y cambia el contenido del elemento ( innerHTML) a "Hola JavaScript":

Ejemplo de uso

<p id="demo">Javascript puede cambiar el contenido HTML</p>

<button type="button"onclick="document.getElementById('demo').innerHTML = Date()">
Click me to display Date and Time.</button>

JavaScript puede cambiar los valores de atributos HTML

En este ejemplo JavaScript cambia el valor de la src(fuente) atributo de un <img> tag:

<button onclick="document.getElementById('myImage').src='img/pic_bulbon.gif'">Enciende la luz</button>

<img id="myImage" src="img/pic_bulbon.gif" style="width:100px">

<button onclick="document.getElementById('myImage').src='img/pic_bulboff.gif'">Apaga la luz</button>

 

JavaScript puede cambiar los estilos HTML (CSS)

Cambiar el estilo de un elemento HTML, es una variante de cambiar un atributo HTML:

Cambia el color

Código

<p id="color">Cambia el color</p>
<button type="button" class="btn btn-outline-danger" onclick="document.getElementById('color').style.color='red'">Rojo</button> <button type="button" class="btn btn-outline-success" onclick="document.getElementById('color').style.color='green'">Verde</button> <button type="button" class="btn btn-outline-primary" onclick="document.getElementById('color').style.color='blue'">Blue</button>
