JavaScript salida

Las posibilidades de visualización de JavaScript

JavaScript puede alterar los datos "Display" de diferentes maneras:

  • La escritura en un elemento HTML, usando  innerHTML.
  • La escritura en la salida HTML utilizando document.write().
  • Escribir en un cuadro de alerta, utilizando  window.alert().
  • Escribir en la consola del navegador, usando  console.log().

 

Usando innerHtml

Para acceder al contenido HTML de un elemento se utiliza innerHTML.

Usando document.write

Para propósitos de prueba se utiliza document.write() para mostrar los datos.

Usando window.alert

Para propósitos informativo utilizar el método window.alert() para mostrar los datos.

Usando console.log

Para propósitos de depuración utilizar el método console.log() para mostrar los datos.

Cuadro de diálogos

Las posibilidades de visualización de JavaScript

 

confirm()

El método confirm () muestra un cuadro de diálogo con un mensaje especificado, junto con un clic en Aceptar y un botón Cancelar.

confirm(message)

alert()

El método alert () muestra un cuadro de alerta con un mensaje especificado y un botón Aceptar.

alert(message)

prompt()

El método prompt() muestra un cuadro de diálogo que solicita al visitante para la entrada de datos, devuelve el valor de entrada si el usuario hace clic en "OK". Si el usuario hace clic en "cancelar" el método devuelve null.

prompt(text, defaultText)


Métodos de objeto window

close()

El método close() cierra una nueva ventana del navegador..

window.close()

open()

El método open() imprime el contenido de la ventana actual.

window.open(URL, name, specs, replace)

print()

El método print() imprime el contenido de la ventana actual.

window.print()

setInterval()

El método setInterval llama a una función o evalúa una expresión a intervalos especificados (en milisegundos).

setInterval(function, milliseconds, param1, param2, ...)
