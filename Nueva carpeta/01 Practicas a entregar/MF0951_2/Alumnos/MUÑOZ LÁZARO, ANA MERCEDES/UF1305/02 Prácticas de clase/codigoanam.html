¿Que podemos hacer con JavaScript?

¿Dónde se inserta JavaScript?

La etiqueta <script>

  • El código JavaScript en HTML debe ser insertado entre <script>y </script>etiquetas.
  • Los scripts pueden ser colocados en la sección <body>, o en el <head> de una página HTML, o en ambos.
  • Los scripts pueden ser colocados en archivos externos com js7funciones.je
  • El código JavaScript puede ser insertado en líneas, en las etiquetas de HTML del del atributo del evento onClick.

Ejemplos de edad de JavaScript pueden utilizar un atributo de tipo: <script type = "text / javascript">. 
No se requiere el atributo type. JavaScript es el lenguaje de scripts por defecto en HTML.

JavaScript en <head>

Código

<!DOCTYPE html>
<html>

<head>
<script>
function myFunction() {
document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</head>
<body>

<h1>A Web Page</h1>
<p id="demo">A Paragraph</p>
<button type="button" onclick="myFunction()">Try it</button>

</body>
</html>

Ejemplo de uso
