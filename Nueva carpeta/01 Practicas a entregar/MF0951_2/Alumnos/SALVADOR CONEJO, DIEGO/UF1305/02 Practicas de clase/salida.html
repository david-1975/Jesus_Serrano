Las posibilidades de visualización de JavaScript

JavaScript puede alterar los datos "Display" de diferentes maneras:

  • La escritura en un elemento HTML, usando innerHTML.
  • La escritura en la salida HTML utilizando document.write().
  • Escribir en un cuadro de alerta, utilizando window.alert().
  • Escribir en la consola del navegador, usando console.log().

Cuadros de diálogos de ventanas

JavaScript puede mostrar cuadros de diálgos de ventanas de diferentes maneras:

  • El método confirm() muestra un cuadro de diálogo con un mensaje especificado.
  • El método alert() muestra un cuadro de alerta con un mensaje especificado y un botón Aceptar.
  • El método prompt() muestra un cuadro de diálogo que solicita al usuario entrada de datos.
  • El método open() abre una nueva ventana del navegador.
  • El método close() cierra una ventana del navegador.
  • El método print() imprime el contenido de la ventana actual.

Usando confirm()

El método confirm() muestra un cuadro de diálogo con un mensaje especificado, con la posibilidad de pulsar Aceptar o Cancelar.

confirm(message)

Usando alert())

El método alert() muestra un cuadro de alerta con un mensaje especificado y un botón Aceptar.

alert(message)

Usando prompt()

El método prompt() muestra un cuadro de diálogo que solicita al usuario entrada de datos.

Un cuadro de mensaje se utiliza a menudo si desea que el usuario introduzca un valor antes de entrar en una página.

prompt(text, defaultText)

Usando open()

El método open() abre una nueva ventana del navegador.

window.open(URL, name, specs, replace)

Usando close()

El método close() cierra la ventana actual..

window.close()

Usando print()

El método print() imprime el contenido de la ventana actual.

El método print() abre el cuadro de diálogo Imprimir, que permite al usuario seleccionar las opciones de impresión preferidos.

window.print()

Usando resizeBy() y resizeTo()

El método resizeBy() cambia el tamaño de una ventana en la cantidad especificada, con relación a su tamaño actual.

El método resizeTo() cambia el tamaño de una ventana a la anchura y altura especificadas.

resizeBy(width, height)





Usando setInterval()

El método setInterval() llama a una función o evalúa una expresión a intervalos especificados (en milisegundos).

setInterval(function, milliseconds, param1, param2, ...)

Usando setTimeout()

El método setTimeout() llama a una función o evalúa una expresión a intervalos especificados (en milisegundos).

setTimeout(function, milliseconds, param1, param2, ...)

Usando btoa() y atob()

El método btoa() codifica una cadena en base-64.

El método atob() descodifica una cadena de base-64 codificada.

window.btoa(str)

window.atob(encodedStr)

Usando moveBy() y moveTo()

El método moveBy() se mueve una ventana de un número especificado de píxeles con respecto a sus coordenadas actuales.

El método moveTo () mueve el borde izquierdo y la parte superior de una ventana a las coordenadas especificadas.

window.moveBy(x, y)

window.moveTo(x, y)
