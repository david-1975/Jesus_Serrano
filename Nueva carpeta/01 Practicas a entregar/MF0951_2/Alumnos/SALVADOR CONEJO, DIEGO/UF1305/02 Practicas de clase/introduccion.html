JavaScript puede cambiar el contenido HTML

Uno de los muchos métodos JavaScript HTML es getElementById().

En este ejemplo se utiliza el método de "encontrar" un elemento HTML (con id = "demo") y cambia el contenido del elemento ( innerHTML) a "Hola JavaScript":

Ejemplo de uso

<p id="demo1">Javascript puede cambiar el código HTML</p>
<button type="button" onClick="document.getElementById('demo1').innerHTML='¡Hola Javascript!'">¡Púlsame!</button>

JavaScript puede cambiar los valores de atributos HTML

En este ejemplo JavaScript cambia el valor de la src(fuente) atributo de una imagen:

Ejemplo de uso

<button onclick="document.getElementById('imagen').src='pic_bulbon.gif'">Enciende la luz</button>
<img src="./img/pic_bulboff.gif" alt="myImagen" id="imagen" style="width: 100px">
<button onclick="document.getElementById('imagen').src='pic_bulboff.gif'">Apaga la luz</button>
