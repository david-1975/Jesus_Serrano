Métodos de Array

Método Descripción Síntaxis  
new Array() Crea un nuevo objeto de tipo Array con los datos existentes en variables. new Array()
join() Método que también se une a todos los elementos de la matriz en una cadena. array.join(separator)
pop() Método que elimina el último elemento de una matriz array.pop()
push() Método que añade un nuevo elemento a una matriz. Lo hace al final de la matriz. array.push(item1, item2, ..., itemX)
shift() Método que elimina el primer elemento de matriz y "desplaza" todos los otros elementos a un índice inferior. array.shift()
unshift() Método que añade un nuevo elemento a una matriz (al principio). array.unshift(item1, item2, ..., itemX)
sort() Método que ordena los elementos de una matriz. array.sort(compareFunction)
length Propiedad que ofrece una forma fácil de añadir un nuevo elemento a una matriz. length
splice() Método que se puede utilizar para añadir nuevos elementos a una matriz.

El primer parámetro (index) define la posición donde se deben nuevos elementos añadidos (empalmados).

El segundo parámetro (howmany) define el número de elementos debe ser eliminado .

 array.splice(index, howmany, item1, ....., itemX)
concat() Método que crea una nueva matriz mediante la fusión (concatenar) matrices existentes. array1.concat(array2, array3, ..., arrayX)
reverse() El método reverse () invierte el orden de los elementos en una matriz. array.reverse()
reduce()

El método reduce() reduce la matriz a un solo valor. Realiza una función proporcionada para cada valor de la matriz (de izquierda a derecha). El valor de retorno de la función se almacena en un acumulador (resultado / total).

 array.reduce(function(total, currentValue, currentIndex, arr), initialValue)
every()

El método every() comprueba si todos los elementos de una matriz cumplen una condición (proporcionado como una función).

El método every() ejecuta la función de una vez para cada elemento presente en la matriz:

  • Si encuentra un elemento de matriz donde la función devuelve un falso valor, every() devuelve falso (y no comprueba los valores restantes)
  • Si no se producen falsas, every() devuelve true
 array.every(function(currentValue, index, arr), thisValue)
slice()

El método slice() devuelve los elementos seleccionados en una matriz, como un nuevo objeto de matriz. Selecciona los elementos a partir de un argumento de inicio (start), y termina en, pero no incluye , el argumento final (end).

 array.slice(start, end)
fill()

El método fill() llena todos los elementos de una matriz con un valor estático. Es posible especificar el índice para iniciar y finalizar fill(). Por defecto se cambia toda la matriz.

 array.fill(value, start, end)
prototype

El constructor prototype le permite añadir nuevas propiedades y métodos al objeto Array ().

Cuando se construye una propiedad, todas las matrices tendrán la propiedad, y su valor, como por defecto.

Cuando se construye un método, todas las matrices tendrán este método disponible.

 Array.prototype.name = value
