¿Dónde se puede insertar Javascript?

  • Se puede insertar en línea, dentro de las propias etiquetas HTML dentro de atributos como onClick, como hemos visto anteriormente.
  • En HTML, el código JavaScript debe ser insertado entre etiquetas <script> y </script>.
  • Los scripts pueden ser colocados en el <body>, o en la <head> sección de una página HTML, o en ambos.
  • Los scripts también pueden ser colocados en archivos externos como js/funciones.js.

Ejemplos antiguos de JavaScript pueden utilizar un atributo de tipo: <script type = "text / javascript">
No se requiere el atributo type. JavaScript es el lenguaje de scripts por defecto en HTML.

Ejemplos de uso

Ejemplo Javascript inmerso en <head>

<!DOCTYPE html>
<html>

<head>
<script>
function myFunction() {
document.getElementById("demo").innerHTML = "Paragraph changed.";
}
</script>
</head>
<body>

<h1>A Web Page</h1>
<p id="demo">A Paragraph</p>
<button type="button" onclick="myFunction()">Try it</button>

</body>
</html>
