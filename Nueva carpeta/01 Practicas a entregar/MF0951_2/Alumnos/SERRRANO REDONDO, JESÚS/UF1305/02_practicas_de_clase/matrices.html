Array (Matriz)

El concepto de matrices

Una matriz es una variable especial, que puede contener más de un valor a la vez. Una matriz puede contener muchos valores bajo un único nombre, y se puede acceder a los valores de referencia a un número de índice. Las matrices usan números para acceder a sus "elementos"

Métodos matrices

Método Descripción Síntasix  
new Array() Crea un nuevo objeto de tipo Array con los datos existentes en variables. new Array()
concat() Se utiliza para unir dos o más matrices. array1.concat(array2, array3, ..., arrayX)
copyWithin() Copia método de los elementos de la matriz dentro de la matriz, hacia y desde las posiciones especificadas. array.copyWithin(target, start, end)
entries() Devuelve un objeto Iterador matriz con pares clave / valor. array.entries()
every() Comprueba si todos los elementos de una matriz pasan una prueba (proporcionado como una función). array.every (function(currentValue, index, arr), thisValue)
fill() Llena todos los elementos de una matriz con un valor estático array.fill(value, start, end)
filter() Crea una matriz llena de todos los elementos de la matriz que pasan una prueba (proporcionado como una función). array.filter (function(currentValue, index, arr), thisValue)
find() Devuelve el valor del primer elemento en una matriz que pasa una prueba (proporcionado como una función). array.find(function(currentValue, index, arr),thisValue)
findIndex() Devuelve el índice del primer elemento de una matriz que pasan una prueba (proporcionado como una función). array.findIndex(function(currentValue, index, arr), thisValue)
forEach() Llama a una función proporcionada una vez por cada elemento de una matriz, en orden. array.forEach(function(currentValue, index, arr), thisValue)
from() Devuelve un objeto Array de cualquier objeto con una propiedad de longitud o un objeto iterable. Array.from(object, mapFunction, thisValue)
includes() Determina si una matriz contiene un elemento especificado. array.includes(element, start)
indexOf() Busca en la matriz para el elemento especificado, y devuelve su posición. array.indexOf(item, start)
isArray() Determina si un objeto es una matriz. Array.isArray(obj)
join() Se une a los elementos de una matriz en una cadena, y devuelve la cadena. array.join(separator)
keys() Devuelve un objeto Iterador matriz con las claves de una matriz. array.keys()
lastIndexOf() Busca en la matriz para el elemento especificado, y devuelve su posición. array.lastIndexOf(item, start)
length Devuelve el número de elementos de una matriz. array.length
map() Crea una nueva matriz con los resultados de llamar a una función para cada elemento de la matriz. array.map(function(currentValue, index, arr), thisValue)
pop() Elimina el último elemento de una matriz, y devuelve dicho elemento. array.pop()
push() Añade nuevos elementos al final de una matriz, y devuelve la nueva longitud. array.push(item1, item2, ..., itemX)
reduce() Reduce la matriz a un solo valor. Realiza una función proporcionada para cada valor de la matriz (de izquierda a derecha). array.reduce(function(total, currentValue, currentIndex, arr), initialValue)
reduceRight() Reduce la matriz a un solo valor. Realiza una función proporcionada para cada valor de la matriz (de derecha a izquierda). array.reduceRight (function(total, currentValue, currentIndex, arr), initialValue)
reverse() Invierte el orden de los elementos en una matriz. array.reverse()
shift() Elimina el primer elemento de una matriz. array.shift()
slice() Devuelve los elementos seleccionados en una matriz, como un nuevo objeto de matriz. array.slice(start, end)
some() Comprueba si alguno de los elementos de una matriz pasan una prueba (proporcionado como una función). array.some(function(currentValue, index, arr), thisValue)
sort() Ordena los elementos de una matriz. array.sort(compareFunction)
splice() Añade / quita artículos a / desde una matriz, y devuelve el elemento (s) eliminado. array.splice(index, howmany, item1, ....., itemX)
toString() Convierte una matriz en una cadena y devuelve el resultado. array.toString()
unshift() Añade nuevos elementos a la comienzo de una matriz, y devuelve la nueva longitud. array.unshift(item1, item2, ..., itemX)
valueOf() Devuelve la matriz. array.valueOf()
