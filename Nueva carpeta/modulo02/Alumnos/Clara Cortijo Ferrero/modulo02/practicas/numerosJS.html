Number Methods

Método Descripción  
isFinite() Comprueba si un valor es un número finito
isInteger() Comprueba si un valor es un entero
isNaN() Comprueba si un valor es NaN
toExponential(x) Convierte un número a notación exponencial
toFixed(x) Formatea un número para que tenga un número especificado dedecimales
toPrecision(x) Formatea un número a una longitud x especificada
toString() Convierte un número a cadena de caracteres
valueOf() Devuelve el valor primitivo de un número

All number methods return a new value. They do not change the original variable.
