Métodos string (cadenas)

El concepto de cadenas

Un string(cadena), es cero o más caracteres escritos entre comillas. Se utiliza como valor en propiedades de objetos, arrays y variables.

Las cadenas JavaScript se utilizan para almacenar y manipular texto. Los métodos son funciones que permiten manipular las cadenas de texto.

Secuencias de escape JavaScript
Código Resultado
\b Espacio en blanco
\f Avance de página
\n Nueva línea
\r Retorno de carro
\t Tabulador horizontal

La secuencia \n inserta una línea nueva en una cadena

Secuencias de escape JavaScript
Código Resultado Descripción
\' ' Comillas simples
\" " Comillas dobles
\\ \ Barra invertida
\u00F1 ñ Letra ñ minúscula
\u00D1 Ñ Letra ñ mayúscula

La secuencia \" inserta una comilla doble en una cadena

Método Descripción  
length() Devuelve la longitud de una cadena
chartAt() Devuelve el carácter en el índice especificado
charCodeAt() Devuelve el índice del caracter Unicode especificado
fromCharCode() Devuelve el caracter asociado a un código Unicode
toUpperCase() Pasa un texto a mayúsculas
toLowerCase() Pasa un texto a minúsculas
concat() Concatena dos cadenas
endsWith() Comprueba si una cadena termina con una cadena de caracteres especificados
includes() Comprueba si una cadena contiene la cadena de caracteres especificados
indexOf() Devuelve la posición de la primera aparición encontrada de un valor especificado en una cadena
lastindexOf() Devuelve la posición de la última aparición encontrada de un valor especificado en una cadena
localeCompare() Compara alfabéticamente dos cadenas
match() Busca una cadena que coincida con una expresión regular, y devuelve las coincidencias separadas por comas
repeat() Devuelve una cadena nueva con un número especificado de copias de una cadena existente
replace() Reemplaza una cadena por otra especificada
search() Busca una expresión regular dentro de una cadena
slice() Extrae un trozo de cadena indicándole el principio y fin
split() Divide una cadena en un array de subcadenas
startsWith() Comprueba si una cadena comienza con los caracteres especificados
substr() Extrae partes de una cadena comenzando en una posición especificada y la longitud de caracteres a mostrar
substring() Extrae partes de una cadena comenzando en una posición inicial especificada y una posición final
toString() Convierte un objeto en una cadena de caracteres
trim() Quita espacios por delante y por detrás, no enmedio
