(Área de un cuadrado)
Calcular el área de un cuadrado o rectángulo de X metros de lado y X metros de alto.
fórmula área cuadrado = base *altura= m2) (ejemplo área = 3 m x 3 m = 9 m2)

(Área de un triángulo)
Calcular el área de un triángulo de X metros de lado y X metros de altura.

(Tiempo de descarga)
Calcular cuánto tiempo tardaría en descargar un archivo de X megabytes con una velocidad de descarga de 10 mbps (megabit por segundo).
(Fórmula: tamaño de archivo / tasa de transferencia = tiempo en segundos) (1*1024*8) / 10) = 819,2 segundos; 819,2 / 60 = 13,65 minutos y segundos.
(Almacenamiento 1024 byte, transferencia 1000 bps)
