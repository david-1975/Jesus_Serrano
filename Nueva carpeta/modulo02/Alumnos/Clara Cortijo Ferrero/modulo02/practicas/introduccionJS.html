JavaScript puede cambiar el contenido HTML

Uno de los muchos métodos JavaScript HTML es getElemenById () .

En este ejemplo se utiliza el método de "encontrar" un elemento HTML (con id = "demo") y cambia el contenido del elemento ( innerHTML ) para "Hola JavaScript":

Código JavaScript

<p class="negrita" id="demo"><u>JavaScript puede cambiar el contenido HTML</u></p>
<button type="button" onClick="document.getElementById('demo').innerHTML='<mark>Hola, JavaScript, puede cambiar el texto</mark>'">onClick usar mejor entre comillas dobles</button>

JavaScript puede cambiar los valores de atributos HTML

En este ejemplo JavaScript cambia el valor del atributo src (fuente) de una etiqueta <img>:

Código JavaScript

<button type="button" onClick="document.getElementById('bombilla').src='img/pic_bulbon.gif'">Enciende la luz</button>
<img id="bombilla" src="img/pic_bulboff.gif" width="100" height="180" alt="bombillaoff"/>
<button type="button" onClick="document.getElementById('bombilla').src='./img/pic_bulboff.gif'">Apaga la luz</button>

JavaScript puede cambiar los estilos HTML (CSS)

Cambiar el estilo de un elemento HTML, es una variante de cambiar un atributo HTML:

Código JavaScript

<button type="button" onClick="document.getElementById('texto').style.color='red'">Cambia a rojo</button>
<button type="button" onClick="document.getElementById('texto').style.color='green'">Cambia a verde</button>
<button type="button" onClick="document.getElementById('texto').style.color='blue'">Cambia a azul</button>
