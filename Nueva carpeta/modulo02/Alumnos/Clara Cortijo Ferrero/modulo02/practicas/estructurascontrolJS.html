Estructuras de control de flujo

Estructura condicional

Método Descripción Sintaxis  
if Ejecuta una serie de instrucciones si se cumple una determinada condición. if(condicion){..}
else Ejecuta una nueva condición si la primera condición no se cumple. if(condicion){..}else{..}
else if Especifica una nueva condición si la primera condición no se cumple. if(condicion){..} else if(condicion){..} else{..}
switch Realiza diferentes acciones basadas en diferentes condiciones.

switch(expresion) {
case a: Bloque de código break;
case b: Bloque de código break;
case z: Bloque de código;
default: Bloque de código;
}

Estructura repetitiva

for Ejecuta un bloque de código tantas veces como valores diferentes haya. for(declaracion1;declaracion2;declaracion3){Bloque de código a ejecutar}
while Ejecuta una nueva condición si la primera condición no se cumple. if(condicion){..}else{..}
do while Especifica una nueva condición si la primera condición no se cumple. if(condicion){..} else if(condicion){..} else{..}
