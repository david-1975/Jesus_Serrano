Las sentencias de JavaScript se componen de: valores, operadores, expresiones, palabras clave y comentarios.

Al igual que el Álgebra

Las variables en JavaScript son contenedores para el almacenamiento de valores de datos

JavaScript utiliza operadores aritméticos (+ - * /) para calcular los valores.

En este ejemplo x, y, z son variables:

Código JavaScript

<p id="demo"></p>
<script>
var x = 10; var y = 20; var z = x + y;
document.getElementById("demo").innerHTML = "El resultado de la variable z es: " + z + ".";
</script>

Ejemplos de variables de JavaScript

<script>
var x = 10; //Declaración de variable
var y = 20; //Declaración de variable
var z = x + y; //Cálculo de variables
</script>

<script>
var x,y,z; //Declaración de variable
x=10; y=20;//Valores de las variables
z = x + y;//Cálculo de variables
</script>

<script>
var x; x=10;//Declaración variable
var y; y=20;//Declaración variable
var z; z = x + y;//Variable y cálculo
</script>
