Métodos de matriz

Una matriz es una variable especial, que puede contener más de un valor a la vez.

Una matriz puede contener muchos valores bajo un único nombre, y se puede acceder a los mismos con el número de índice (=posición).

Métodos Descripción  
split() Convierte una cadena en una matriz de datos
toString() Convierte una matriz en un string o cadena de texto
join() Une todos los elementos de un array en un string
pop() Elimina el último elemento de la matriz
push() Añade un nuevo elemento a una matriz
shift() Elimina el primer elemento de la matriz y desplaza el resto de elementos un índice menos
unshift() Añade un nuevo elemento al principio de una matriz y desplaza el resto de elementos un índice más
splice() Añade o remueve nuevos elementos a una matriz
concat() Concatena varias matrices
slice() Elimina uno o varios elementos de una matriz
sort() / reverse() Ordena una matriz alfabéticamente de forma ascendente, o descendente usando reverse
