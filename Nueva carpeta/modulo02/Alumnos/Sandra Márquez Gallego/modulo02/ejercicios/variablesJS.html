Al igual que Álgebra

Las variables de JavaScript son contenedores para el almacenamiento de valores de datos.

JavaScript utiliza operadores (+-*/) aritméticos para calcular los valores.

Código JavaScript

<p id="demo"></p>
<script>
var x=10; var y=20; var z=x+y;
document.getElementById("demo").innerHTML = "El resultado de la variable z es:" + " " + z + " . ";
</script>
