Método de matriz

Una matriz es una variable especial, que puede contener más de un valor a la vez.

Una matriz puede contener muchos valores bajo un único nombre, y se puede acceder a los valores de referencia a un número de índice (posición).

Método Descripción  
split ( ) Convierte una cadena en una matríz de datos.
toString( ) Convierte una matriz a una cadena separadas por comas.
join( ) Une todos los valores de una matríz en un string con un separador.
pop( ) Elimina el último elemento de una matriz.
push ( ) Añade un nuevo elemento a una matriz en la posicón final de la matríz.
shift( ) El cambio() método elimina el primer elemento de la matriz y "desplaza" todos los otros elementos a un índice inferior.
unshift( ) Añade un nuevo elemento a una matriz al principio,
unshift( ) Añade o remueve nuevos elementos a una matriz.
concat( ) Concatena los datos de distintas variables.
slice( ) Extrae una parte del array.
sort( ) Ordena en orden alfabético la matriz.
reverse( ) Ordena de forma inversa la matriz.
sort( )reverse( ) Ordena de forma alfabeticamente y luego invierte el orden.
