Métodos numéricos

Método Descripción
isFinite() Comprueba si un valor es un número finito.
isInteger() Comprueba si un valor es un número entero.
isNaN() Comprueba si un valor no es un número.
toExponential(x) Convierte a un número exponencial.
toFixed(x) Formatea un número con x digitos después del final.
toPrecision(x) Formats a number to x length
toString() Converts a number to a string
valueOf() Returns the primitive value of a number

Todos los métodos numéricos devuelven un valor nuevo. No cambian la variable original.
