Métodos de fechas

Estos métodos se pueden utilizar para obtener información de un objeto de fecha:

Method Description
getDay() getDay() devuelve el día de la semana de una fecha como un número (0-6):
getFullYear() getFullYear () devuelve el año de una fecha como un número de cuatro dígitos:
getMonth() El getMonth () devuelve el mes de una fecha como un número (0-11):
getDate() El getDate () devuelve el día de una fecha como un número (1-31):
getHours() Los getHours () devuelve el hora de una fecha como un número (0-23):
getMinutes() Los getMinutes () devuelve el minuto de una fecha como un número (0-59):
getSeconds() Los getSeconds () devuelve el segundo de una fecha como un número (0-59):
getMilliseconds() Los getMilliseconds () devuelve los milisegundos de una fecha como un número (0-999):
getTime() El getTime () devuelve el número de milisegundos desde 1 de enero de 1970:
