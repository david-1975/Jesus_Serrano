Métodos string de cadenas

El concepto de cadenas

Un string (cadena), es cero o maás caracteres escritos entre comillas. Se utliza como valor en propiedades de objetos, matrices y variables.

Las cadenas JavaScript se utilizan para almacenar y manipular texto. Los métodos son funciones que permiten manipular las cadenas de texto.

Método Descripción  
length() Devuelve la longitud de una cadena
charAt() Devuelve el caracter especificado por un valor
charCodeAt() Código UNICODE de un caracter (ASCII)
fronCharCode() Caracter (ASCII) de un Código UNICODE
toUpperCase() Convertir a mayusculas
toLowerCase() Convertir a minusculas
concat() Unir cadenas
endsWith() Conprueba si una cadena termina con una cadena/texto especificado
includes() Conprueba si una cadena incluye una cadena/texto especificado
indexOf() Busca la posicion en la que comienza una cadena
lastIndexOf() Busca la posicion en la que comienza la última ocurrencia de una cadena
localeCompare() compara 2 cadenas alfabeticamente
match() busca una cadena que coincida con una expresion regular y devuelve las ocurrencias separadas por comas
repeat() repite cadena n veces
replace(cadena1, cadena2) reemplaza una cadena
search() Busca una cadena o expresion regular y devuelve su posición o -1
slice() extrae partes de una cadena desde-hasta
split() Divide una cadena en un array de subcadenas
