Métodos de Matriz

Una matriz es una variable especial, que puede contener más de un valor a la vez.

Una matriz puede contener muchos valores bajo un único nombre, y se puede acceder a los valores de referencia a un número de índice.

Metodo Description exex
split() Convierte una cadena en una matriz de datos
toString() Convierte una matriz de datos en una cadena
join() une los valores de un array/matriz en un string con separadores
pop() El pop () método elimina el último elemento de una matriz
push() El push método añade un elemento a la matriz
shift() El shift() método elimina el primer elemento de una matriz
unshift() El método shift añade un elemento al principio de la matriz

Todos los métodos numéricos devuelven un valor nuevo. No cambian la variable original.
