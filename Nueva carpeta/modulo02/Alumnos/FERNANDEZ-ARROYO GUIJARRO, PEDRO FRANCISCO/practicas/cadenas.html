Métodos string de cadenas

El concepto de cadenas

Un string (cadena), es cero o maás caracteres escritos entre comillas. Se utliza como valor en propiedades de objetos, matrices y variables.

Las cadenas JavaScript se utilizan para almacenar y manipular texto. Los métodos son funciones que permiten manipular las cadenas de texto.

Secuencias de escape JavaScript
Código Resultado
\b Espacio en blanco
\f Avance de página
\n Nueva línea
\r Retorno de carro
\t Tabulator Horizontal

La secuencia \n inserta una línea nueva en una cadena

Secuencias de escape JavaScript
Código Resultado Descripción

\" " Comillas dobles
\\ \ Barra invertida
\u00F1 ñ Letra ñ minúscula
\u00D1 Ñ Letra ñ mayúscula

La secuencia \" inserta una comilla doble en una cadena
