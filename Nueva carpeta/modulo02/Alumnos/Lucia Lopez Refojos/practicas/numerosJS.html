Métodos numéricos

Método Descripción  
isFinite() Comprueba si un valor es un número finito.
isInteger() Comprueba si un valor es un entero.
isNaN() Comprueba si un valor no es un número NaN
toExponential(x) Convierte un número en valor exponencial
toFixed(x) Formatea un número con x números de dígitos después del punto decimal
toPrecision(x) Formatea un número
toString() Converts a number to a string
valueOf() Returns the primitive value of a number

Todos los métodos numéricos devuelven un valor nuevo. No cambian la variable original.
