Métodos de matrices JavaScript

Una matriz es una variable especial, que puede contener más de un valor a la vez.

Una matriz puede contener muchos valores bajo un único nombre, y se puede acceder a los valores de referencia a un número de índice (posición).

Método Descripción  
split() Convierte una cadena en una matriz de datos.
toString() Convierte una matriz a una cadena separada por comas.
join() Une los valores de una matriz en un string con los sepadores seleccionados.
pop() Elimina el último elemento de una matriz y lo sacamos mediante alert.
push() Añade un nuevo elemento a una matriz (al final).
shift() Elimina el primer elemento de la matriz y "desplaza" todos los otros elementos a un índice inferior.
unshift()  Añade un nuevo elemento a una matriz (al principio).
splice() Añade o elimina nuevos elementos a una matriz
concat() Crea una nueva matriz fusionando (concatenando) las matrices existentes;
slice() Corta un trozo de una matriz en una nueva matriz.
sort() Ordena una matriz alfabéticamente
reverse() Ordena de manera inversa la matriz
