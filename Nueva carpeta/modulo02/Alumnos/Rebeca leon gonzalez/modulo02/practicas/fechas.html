Estos métodos se pueden utilizar para obtener información de un objeto de fecha:

Method Description
new Date ( ) Coge la fecha actual del sistema.
getFullYear() Coge el año (yyyy)
getMonth() Coge el mes de (0-11)
getDate() Coge el  dia  del numero (1-31)
getHours() Coge la  hora (0-23)
getMinutes() Coge los  minutos (0-59)
getSeconds() Coge los  segundos (0-59)
getMilliseconds() Get the milisegundos (0-999)
getTime() Coge la hora (milisegundos desde Enero 1, 1970)
getDay() Coge el numero del dia de la semana (0-6)
