Metodos String (cadenas) Javascript

Longitud de una cadena de texto

Un String (cadena), es cero o más caracteres escritos entre comillas. Se utiliza como valor en propiedades de objetos, arrays y variables.

Las cadenas JavaScript se utilizan para almacenar y manipular texto. Los métodos son funciones que permiten manipular las cadenas de texto.

La propiedad length devuelve la longitud de una cadena.

Secuencias de escape JavaScript
Código Resultado
\b Espacio en blaco
\f Avance de pagina
\n Nueva linea
\r Retorno de carro
\t Tabulador Horizontal

La secuencia \n inserta una línea nueva en una cadena

Secuencias de escape JavaScript
Código Resultado Descripción
\' ' Comillas simples
\" " Comillas dobles
\\ \ Barra invertida
\u00F1 ñ Letra ñ minúscula
\u00D1 Ñ Letra Ñ mayúscula

La secuencia \" inserta una comilla doble en una cadena

Método Descripción Ejecutables
length( ) Devuelve la longitud de una cadena.
charAt ( ) Devuelve el carácter en el índice especificado (posición)
charCodeAt ( ) Devuelve índice del carácter Unicode especificado.
fromCharCode ( ) Convierte los valores Unicode en caracteres.
toUpperCase( ) Convierte a mayúsculas
toLowerCase( ) Convierte a minúsculas
concat ( ) Une dos o más cadenas y devuelve una nueva cadena unida.
endsWith ( ) Comprueba si una cadena termina con una cadena o caracteres especificados.
includes ( ) Comprueba si una cadena termina con una cadena o caracteres especificados.
indexOf ( ) Devuelve la posición de la primera aparición encontrada de un valor especificado en una cadena.
lastIndexOf ( ) Devuelve la posición de la última aparición encontrada de un valor especificado en una cadena.
localeCompare( ) Compara dos cadenas en la localización actual.
match ( ) Busca una cadena que coincida con una expresión regular y devuelve las coincidencias separadas por comas.
repeat () Devuelve una cadena nueva con un número especificado de copias de una cadena existente.
replace( ) Busca una cadena con una expresión regular y devuelve una cadena sustituyendo los valores especificados.
search( ) Busca una cadena para un valor especificado o una expresión regular y devuelve la posición de la coincidencia.
slice ( ) Extrae parte de una cadena y devuelve una nueva cadena, con la parte extraída.
split ( ) convierte una cadena de texto en un array.
startsWith( ) Comprueba si una cadena empieza con una cadena / caracteres especificados
substr( ) Extrae partes de una cadena comenzando en una posición especificada y añadiendo la longitud de caracteres a mostrar.
substring( ) Extrae los caracteres de una cadena entre dos indices especificados. (Posición).
toString( ) Convierte cualquier objeto en una cadena de carácteres.
trim ( ) Elimina el espacio en blanco por delante y por detras de una cadena de texto( string).
