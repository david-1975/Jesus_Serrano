Extructura condicional

El concepto de control de flujo

Método descripcion sintaxis  
if Ejecuta una serie de instrucciones si se cumple una determinada condición. if (condicion){...}
else Ejecuta una nueva condicion si la primera condicion no se cumple. if ( condicion ) else{..}
else if Especifica una nueva condicion si la primera condicion no se cumple if (condicion) else if(condicion) else{..}
switch Realiza diferentes acciones basadas en diferentes condiciones . switch(expresión){
caso a : bloque de código break;
caso b :bloque de código break;
caso c : bloque de código;
caso d : bloque de código
default : bloque de código
}
