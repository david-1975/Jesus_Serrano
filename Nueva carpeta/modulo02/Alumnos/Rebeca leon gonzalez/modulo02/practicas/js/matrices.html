Métodos de matriz(array)

Una matriz es una variable especial, que puede contener más de un valor a la vez.

Una matriz puede contener muchos valores bajo un solo nombre, y puede acceder a los valores de referencia con el número de índice (posicion).

Método Descripcion  
split ( ) Convierte una cadena en una matríz de datos
toString( ) Convierte una matriz a una cadena (separados por comas).
join ( ) Une los valores de una matríz en un string con un separador que elijas.
pop ( )  Elimina el último elemento de una matriz y lo muestra
push () Añade un elemento solo al final de la matriz.
shift () Elimina el primer elemento de la matriz y "desplaza" todos los demás elementos a un índice más bajo.
unshift ( ) Añade un nuevo elemento a una matriz (al principio), y "desvía" elementos antiguos.
