Variables de JavaScript

Al igual que Álgebra

Las variables en javascript son contenedores para el almacenamiento de datos

Código Javascript

En este ejemplo x, y, z son variables:
<script>
var x = 10;
var y =20;
var z = x + y;

document.getElementById("demo").innerHTML="El resultado de la variable z es: " + z+".";
</script>

Ejemplos de variables Javascript

<script>
var x=10 //Declaración de una variable
var y=20 //Declaración de una variable
var z = x + y; //Cálculo de variables
</script>

<script>
var x,y,z//Declaración de las variables
x=10;y= 20//Declaración de las variables
z = x + y;//Cálculo de variable
</script>

<script>
var x;x= 1;//Cálculo de variable
var y;y= 20; //Cálculo de variable
var z;z= x + y;// Variable y cálculo
</script>
