Métodos de matriz

Método Descripción  
split() Convierta una cadena( string ) en una matriz de datos( array )
toString() Convertir un array en una cadena separado por comas
join() Unirse a los elementos de una matriz en una cadena con un separador
pop() Elimina el último elemento de una matriz, y devuelve dicho elemento.
push() Añade un nuevo elemento al final de una matriz.
shift() Elimina el primer elemento de una matriz y cambia el valor a un indice inferior
unshift() Añade nuevos elementos a la comienzo de una matriz, y cambia el valor a un indice superior
Index number( ) Añade nuevos elementos a la comienzo de una matriz, y cambia el valor a un indice superior
Splice( ) Para añadir nuevos elementos a una matriz. el primer parametro añade un elemento, el segundo, los elementos a borrar
Concat( ) Crea una nueva matriz mediante la fusión (concatenar) matrices existentes
Slice( ) Extrae el elemento o elementos del array con parametros simple o dobles
Sort( ) Ordena los valores como cadenas en orden alfabético y ascendente
Reverse( ) Ordena los valores como cadenas en orden inverso a como las recibe
