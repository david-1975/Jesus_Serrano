Métodos number (numéricos)

El concepto de números

Un número (number) es un tipo de dato. Los números pueden ser escritos con o sin decimales y son precisos hasta los 15 dígitos.

Los métodos numéricos devuelven un valor nuevo y no cambian la variable original.

Todos los métodos numéricos se pueden utilizar en cualquier tipo de números (literales, variables o expresiones).

Métodos numéricos

Método Descripción  
isFinite ( ) Comprueba si un valor es un número finito.
isInteger ( ) Comprueba si un valor es un entero.
isNaN ( ) Comprueba si un valor no es un número (NaN).
toExponential (x) Convierte un número en un número exponencial.
toFixed (x) Formatea un número con x números de dígitos después del punto decimal
toPrecision (x) Formatea un número a una longitud.
toString ( ) Convierte un número en una cadena
valueOf ( ) Devuelve el valor primitivo de un número.
