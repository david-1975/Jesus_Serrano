Operadores de Cálculo de JavaScript

Los operadores aritméticos de JavaScript

Los operadores aritméticos se utilizan para realizar operaciones aritméticas con números:
Operador Descripción Resultado
+ Suma z = x+y  
- Resta z = x-y  
* Multiplicar z = x*y  
/ Dividir z = x/y  
% Resto z = x % y  
++ Incrementar z = x++y  
-- Decrementar z = x--y
