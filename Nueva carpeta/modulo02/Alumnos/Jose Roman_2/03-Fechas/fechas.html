Cálculos de fechas con JavaScript

Edad

Introducir el año de nacimiento y calcular la edad.


Año

Introducir el año y mostrar el año anterior, el año introducido y el año posterior


Fecha

A partir de 3 campos de formulario, día, mes y año, mostrar la fecha en este formato "dd/mm/aaaa".





Hora

A partir de 3 campos de formulario, hora, minutos y segundos, mostrar la hora en este formato "hh:mm:ss".





Formato

Introducir una fecha y mostrarla en este formato: "Hoy es martes, 11 de Noviembre de 2018".



Formato actual

Mostrar la fecha y hora actual en este formato: "Has iniciado sesión el día 13/11/2018 a las 15:30:25".


Días cumpleaños

Calcular los días que faltan para tu cumpleaños a partir de una fecha de nacimiento introducida.


Vida

Calcular cuántos días, horas, minutos y segundos ha vivido una persona a partir de su fecha de nacimiento.


Trimestre

Mostrar a que trimestre pertenece una fecha introducida.


Zodiaco

Mostrar a que trimestre pertenece una fecha introducida.
