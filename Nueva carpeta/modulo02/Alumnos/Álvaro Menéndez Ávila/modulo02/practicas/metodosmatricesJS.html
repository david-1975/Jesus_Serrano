El concepto de matriz

Una matriz es una variable especial, que puede contener más de un valor a la vez.

Una matriz puede contener muchos valores bajo un único nombre, y se puede acceder a ellos mediante el número de índice.

Métodos de matriz

Método Descripción Ejecución
split() Convierte una cadena en una matriz de datos.
toString() Convierte una matriz a una cadena (separada por comas).
join() Une todos los valores de una matriz en un string con un separador.
pop() Elimina el último elemento de la matriz.
push() Añade un nuevo elemento a una madriz en la posic1ón final.
shift() Elimina el primer valor. No es necesario crear una nueva variable, simplemente reusar la matriz principal.
unshift() Introduce un valor en la primera posición. No es necesario crear una nueva variable, simplemente reusar la matriz principal.
splice() Añade o elimina nuevos elementos a una matriz.
concat() Concatena 2 o más matrices para formar una sola.
slice() Extrae un dato a través de una matriz.
sort() Ordena de forma ascendente los valores de una matriz, alterando todos los índices.
reverse() Ordena de forma inversa los valores de una matriz, alterando todos los índices.
