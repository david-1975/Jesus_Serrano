El concepto de números

Un string (cadena) es cero y más caracteres escritos entre comillas.

Se utiliza como valor en propiedades de objetos, awways y variables

Las cadenas JavaScript se utilizan para almacenar y manipular texto.

Los métodos son funciones que permiten manipular las cadenas de texto.

Métodos numéricos

Método Descripción Ejecución
isFinite() Comprueba si un valor es un número finito.
isInteger() Comprueba si un valor es un número entero.
isNaN() Comprueba si un vlor no es un número (NaN).
toExponential(x) Comprueba si un número es una notación exponencial.
toFixed(x) Formatea un número con x números de dígitos después del punto decimal.
toPrecision(x) Fomatea un número sin redondearlo, solamente muestra dígitos que tú le digas.
toString() Convierte un número en cadena de texto.
valueOf() Devuelve el valor primitivo de un número

All number methods return a new value. They do not change the original variable.
