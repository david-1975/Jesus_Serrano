El BOM (Modelo de Objetos del Nagegador)

El Modelo examinador de objetos (BOM) permite Javascript para "hablar con" el navegador.

El Browser Object Model o BOM, permite acceder y modificar las propiedades de las ventanas del propio navegador. Mediante BOM, es posible redimensionar y mover la ventana del navegador, modificar el texto que se muestra en la barra de estado y realizar muchas otras manipulaciones no relacionadas con el contenido de la página HTML.

No existen normas oficiales para el Browser Object Modelo (BOM). El mayor inconveniente de BOM es que, al contrario de lo que sucede con DOM, ninguna entidad se encarga de estandarizarlo o definir unos mínimos de interoperabilidad entre navegadores.

Objetos de Pantalla

 

Método Descripción  
screen.width , screen.height Ancho y alto de la pantalla
screen.availWidth, screen.availHeight Ancho y alto de de la pantalla disponible
screen.colorDepth, screen.pixelDepth Profundidad de color y píxeles de la pantalla

Objetos de ventana

Método Descripción  
window.location.href Devuelve el href (URL) de la página actual.
window.location.hostname Devuelve el nombre de dominio del proveedor de alojamiento web.
window.location.pathname Devuelve la ruta y el nombre de la página actual.
window.location.protocol  Devuelve el protocolo web utilizado (http: o https :).
window.location.assign Carga un nuevo documento.
window.location.port Devuelve el número del puerto de host a Internet (de la página actual).

Historial

Método Descripción  
window.history.back() Avanza a la página anterior visitada.
window.history.forward() Avanza a la página posterior visitada.

Añadir o eliminar elementos HTML

Método Descripción  
document.createElement(element) Crea un elemento HTML
document.removeChild(element) Elimina un elemento HTML
document.appendChild(element) Añade un elemento HTML
document.replaceChild(element) Reeplaza un elemento HTML
document.write(text) Escribe en el flujo de salida HTML

Cajas emergentes
Método Descripción  
window.alert("texto"); Muestra un cuadro de alerta y el usuario tendria que hacer click en "Aceptar" para continuar.
window.confirm("texto"); Muestra un cuadro de confirmación y el suusairio tendría que verificar o aceptar algo.
window.prompt("texto, defaultText"); Muestra un cuadro con un mensaje para que el usuario introduzca un valor.
