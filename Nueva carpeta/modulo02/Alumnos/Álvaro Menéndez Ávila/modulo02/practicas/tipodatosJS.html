El concepto de tipos de datos

Las variables de JavaScript pueden contener varios tipos de datos: números, cadenas, objetos, matrices, y más

En la programación, tipos de datos es un concepto importante, que permiten operar con variables.

Sin tipos de datos, su ordenador no puede resolver con seguridad esto:

JavaScript String

Una cadena (o una cadena de texto) es uno o más caracteres escritos entre comillas simples o dobles. Las cadenas JavaScript se utilizan para almacenar y manipular texto.

Se puede usar comillas simples dentro de una cadena, siempre y cuando nop coincidan con las comillas dobles que rosdean la cadena.

Ejemplo

Ejemplo de cadenas de texto

var coche = "' '";
var coche = "Volvo";
var coche = "Volvo XC60"
var coche = "Volvo 'modelo' XC60"
var coche = "Volvo \"modelo"\ XC60"

document.getElementById("demo")innerHTML = typeof (coche);

JavaScript Number

JavaScript tiene sólo un tipo de número. Los numeros se pueden escribir con o sin decimales. El hecho de tener letras no implica que deje de ser dato numérico, las cifras hexadecimales usan letras por ejemplo.

Los decimales se separan con un punto, y no con una coma. Los números se escriben sin comillas.

Ejemplos de datos numéricos

var numerico = 3;
var numerico = 3.14;
var numerico = 123e5;
var numerico = 123e-5;
var numeral = 0xFF;

document.getElementById("demo")innerHTML = typeof (numerico);

Ejemplos de datos numéricos

var numeral = "10" - "5";
var numeral = "10" / "5";
var numeral = "10" *"5"
var numeral = (0.2 * 10) / 10;
var numeral = (0.2 * 10 + 0.1 * 10) / 10;

document.getElementById("demo")innerHTML = typeof (numeral);

Typeof = Booleans, Objects, Functions

  • Los datos booleanos se utilizan para realizar comparativa entre datos y pueden tener dos valores; true (verdadero) o false (falso).
  • Los objetos de datos se escriben con llaves y las propiedades del objeto se separan por comas. Las propiedades y sus valores se separan con doble punto, o dos puntos.
  • Las matrices de datos se escriben con corchetes ty los elementos de la matriz estan separados pir comas. Son variables con multiples datos.
  • Una función es un bloque de cñodigo diseñado para realizar una tarea en particular que se ejecuta cuando se la invoca (llama).
