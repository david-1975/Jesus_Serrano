Métodos de fecha

Estos métodos se pueden utilizar para obtener información de un objeto de fecha:

Method Description Ejecución
new Date() Coge la fecha actual del sistema. Puedes especificar una fecha específica entre los paréntesis, no siempre tiene que mostrar la fecha actual del sistema.
getFullYear() Recoge el año como un número de cuatro dígitos (yyyy).
getMonth() Recoge el mes como un número entre el 0 y el 11.
getDate() Recoge el día como un número entre el 1 y el 31.
getHours() Recoge la hora como un número entre el 0 y el 23.
getMinutes() Recoge el minuto como un número entre el 0 y el 59.
getSeconds() Recoge el segundo como un número entre el 0 y el 59.
getMilliseconds() Recoge los milisegundos como un número entre el 0 y el 9999.
getTime() Recoge todos los milisegundos transcurridos desde Enero de 1970).
getDay() Recoge un día de la semana como un número del 0 al 6.
getWeek() Recoge el número de semanas que han transcurrido durante el año.
