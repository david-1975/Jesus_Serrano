Estructuras condicionales

Las sentencias condicionales se utilizan para llevar a cabo diferentes acciones basadas en distintas condiciones.

Método Descripción Sintaxis Ejecución
if Ejecuta una serie de instrucciones si se cumple una determinada condición.

if (condición) {Bloque de código}
else Ejecuta una nueva condición si la primera no se cumple. if (condición) {...} else {...}
else if Especifica una nueva condición si la primera condición no se cumple id (confición) {...} else if (condición) {...} else {...}
switch Realiza diferentes acciones basadas en diferentes condiciones.

switch (expresión) {
case 0: Bloque de código break;
case 1: Bloque de código break;
case 2: Bloque de código break;
default: Bloque de código
