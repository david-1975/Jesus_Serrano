El concepto de cadenas

Un string (cadena) es cero y más caracteres escritos entre comillas.

Se utiliza como valor en propiedades de objetos, awways y variables

Las cadenas JavaScript se utilizan para almacenar y manipular texto.

Los métodos son funciones que permiten manipular las cadenas de texto.

Secuencias de escape JavaScript
Código Resultado
\b Espacio en blanco
\f Avance de página
\n Nueva línea
\r Retorno de carro
\t Tabulator Horizontal

La secuencia \n inserta una línea nueva en una cadena

Secuencias de escape JavaScript
Código Resultado Descripción
\' ' Comillas simples
\" " Comillas dobles
\\ \ Barra invertida
\u00F1 ñ Letra ñ minúscula
\u00D1 Ñ Letra ñ mayúscula

La secuencia \" inserta una comilla doble en una cadena

Métodos String (cadenas)

Método Descripción Ejecución
lenght() Devuelve la longitud de una cadena.
charAt() Devuelve el carácter en el índice especificado (posición).
charCodeAt() Devuelve el índice del caracter Unicode especificado.
fromCodeChar() Devuelve el índice de un código Unicode en caracter.
toUpperCase() Devuelve el texto escrito en mayúsculas.
yoLowerCase() Devuelve el texto escrito en minúsculas.
concat() Devuelve dos valores concatenados.
endsWith() Comprueba si una cadena termina con un texto o caracter especificado. De ser así, la máquina devolverá "true".
includes() Comprueba si una cadena contiene el texto o caracter especificado.
indexOf() Devuelve la posición de la primera aparición encontrada de un valor especificado en una cadena.
lastIndexOf() Devuelve la posición de la última aparición encontrada de un valor especificado en una cadena.
localCompare() Compara dos cadenas siguiendo un criterio del órden alfabético de la página actual.
match() Busca una cadena que coincida unna expresion regular y devuelve las coincidencias separadas con comas.
repeat() Devuelve una cadena nueva con un numero de especificado de copias de una cadena existente.
replace() Busca una cadena con una expresión regular y devuelve una nueva cadena sustituyendo los valores especificados.
search() Busca una cadena para un valor especificado o una expresión regular y devuelve la posición de la coincidencia.
slice() Extrae parte de una cadena y devuelve una nueva cadena con la parte extraída.
split() Divide una cadena en una matriz de subcadenas.
startsWith() Comprueba su una cadena comienza con los caracteres de una cadena específica.
substr() Extrae los caracteres de una cadena, comenzando en una posición especificada, y la longitud de caracteres a mostrar.
substring() Extrae los caracteres de ima cadena entre dos índices especificados (posición).
trim() Elimina el espacio en blanco de ambos extremos de una cadena.
toString() Convierte un objeto en una cadena de caracteres.
