Estructuras de control de flujo

El concepto de Estructuras de control de flujo

These methods can be used for getting information from a date object:

Método Descripción Sintaxis  
if Ejecuta una serie de instrucciones si se cumple una determinada condición. if (condición) {...}
else Ejecuta una nueva condición si la primera condición no se cumple. if(condición) {...} else{...}
else if Especifica una nueva condición si la primera condición no se cumple. if(condición){...} else if(condición) {...} else(...)
switch Realiza diferentes acciones basadas en diferentes condiciones.
switch(expresión) {
case a: Bloque de código break;
case b: Bloque de código break;
case z: Bloque de código  
default: Bloque de código  
}
