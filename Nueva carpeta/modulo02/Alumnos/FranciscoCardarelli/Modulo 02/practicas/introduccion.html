JavaScript Can Change HTML Content

One of many JavaScript HTML methods is getElementById().
This example uses the method to "find" an HTML element (with id="demo") and changes the element content (innerHTML) to "Hello JavaScript":

JavaScript Can Change HTML Content

JavaScript Can Change HTML Content

JavaScript puede cambiar el contenido HTML!

Código JavaScript

<p id="demo">JavaScript Can Change HTML Content</p>
<button type="button" onClick='document.getElementById("demo").innerHTML = "¡HOLA JavaScript!"'>Púlsame</button>


JavaScript Can Change HTML Attribute Values

In this example JavaScript changes the value of the src (source) attribute of an <img> tag:

Codigo JavaScript

<button type="button" onClick="document.getElementById('bombilla').src='img/pic_bulbon.gif' ">Enciende la Bombilla</button>
<img id="bombilla" src="img/pic_bulboff.gif" width="100" height="180" alt="apagado"/>
<button type="button" onClick="document.getElementById('bombilla').src='img/pic_bulboff.gif' ">Apaga la Bombilla</button>


JavaScript Can Change HTML Styles (CSS)

Changing the style of an HTML element, is a variant of changing an HTML attribute:

JavaScript puede cambiar el estilo CSS

Codigo JavaScript

<h3 id="texto">JavaScript puede cambiar el estilo CSS</h3>

<button type="buttton" onClick="document.getElementById('texto').style.color='red' ">Cambiame a Rojo</button>
<button type="buttton" onClick="document.getElementById('texto').style.color='blue' ">Cambiame a Azul</button>
<button type="buttton" onClick="document.getElementById('texto').style.color='green' ">Cambiame a Verde</button>
<button type="buttton" onClick="document.getElementById('texto').style.color='#AA2A2C' ">Cambiame a #AA2A2C</button>
