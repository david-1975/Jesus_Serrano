Metodos string (cadenas) Javascript

El concepto de cadenas

Un string (cadena), es cero o más caracteres escritos entre comillas. Se utiliza como valor en propiedades de objetivos, arrays y variables.
Las cadenas JavaScript se utilizan para almacenar y manipular texto. Los métodos son funciones que permiten manipular las cadenas de texto.


Secuensias de escape JavaScript

Codigo

La secuensia \n inserta una linea nueva en una cadena.

Método Descripción  
length() Devuelve la longitud de una cadena.
charAt() Devuelve el carácter en el índice especificado (posición)
charCodeAt() Devuelve el índice del carácter Unicode especificado
fromcharCode() Convierte el valores Unicode a carácteres.
toUpperCase() Convierte una cadena de texto a mayúsculas
toLowerCase() Convierte una cadena de texto a minúsculas
concat() Une dos o más cadenas y devuelve una cadena unida
endsWith() Comprueva si una cadena termina con una cadena/caracteres especificados.
includes() Comprueva si una cadena contiene la cadena/caracteres especificados.
indexOf() Devuelve la posición de la primera aparación encontrada en un valor especificado en una cadena.
lastIndexOf() Devuelve la posición de la última aparación encontrada en un valor especificado en una cadena.
localeCompare() Compara dos cadenas siguiendo el orden alfabético en la localización actual.
match() Busca una cadena que coincida con una expresión regular y devuelbe las coincidencias separadas por comas.
repeat() Devuelbe una cadena nueva con un número especificado de copias de una cadena existente.
replace() Busca una cadena con una expresión regular y devuelve una nueva cadena sustituyendo los valores especificados.
search() Busca una cadena un valor especificado o una expresión regular y devuelve la posición que coincida.
slice() Extrae parte de una cadena y devuelve una nueva cadena con la parte extraída.
split() Divide una cadena en una matriz de subcadenas.
startsWith() Comprueba si una cadena empieza con una cadena/caracteres especificados.
substr() Extrae parte de una cadena, comenzando en una posición especificada y la longitud de caráteres a mostrar.
substring() Extrae los carácteres de una cadena entre dos índices (posición) especificados.
toString() Convierte un objeto en una cadena de caráteres.
