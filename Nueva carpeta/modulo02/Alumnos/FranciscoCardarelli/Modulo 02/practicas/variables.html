Expresión de las variables

  • Una expresión es una combinación de valores, variables y operadores, que calcula a un valor.
  • Las sentencias de JavaScript se componen de: valores, operadores, expresiones, palabras clave y comentarios.

Al igual que Álgebra

Las variables JavaScript son contenedores para el almacenamiento de valores de datos.
JavaScript utiliza operadores aritméticos ( + - * / ) para calcular los valores.

<p id="demo"></p>
<script>
var x = 15;  //Cálculo de una variable.
var y = 25;  //Cálculo de una variable.
var z = x + y;  //Cálculo de variables.
</script>

<p id="demo"></p>
<script>
var x,y,z;  //Declaración de las variables.
x = 15; y = 25;  //Valores de las variables.
z = x + y;  //Cálculo de variables.
</script>

<p id="demo"></p>
<script>
var x,x = 15;  //Declaración de variables.
var y;y = 25;  //Declaración de variables.
var z;z = x + y;   //Variable y cálculo.
</script>

Código JavaScript

<p id="demo"></p>
<script>
var x = 15; var y = 25; var z = x + y;
document.getElementById("demo").innerHTML = "El resultado de la variable z es:" + " " + z + ".";
</script>


JavaScript es sensible a mayúsculas

Todos los identificadores de JavaScript empiezan con minúculas y pueden contener mayúsculas y minúsculas. 

Las variables lastName y lastname, son dos variables diferentes.

Código JavaScript

</script>
